##### shra rn
- rn /= 2
- c: сюда помещается вытолкнутый бит
- v: не меняется 
- z: если результат равен 0
- n: если старший бит равен 1
##### shla rn
- rn += rn
- c: сюда помещается вытолкнутый бит
- v: 1 если произошло знаковое переполнение
- z: если результат равен 0
- n: если старший бит равен 0
##### shr rn
- rn >>= 1
- c: сюда помещается вытолкнутый бит 
- v: 0
- z: если результат равен 0
- n: если старший бит равен 0
##### shl rn
- rn <<= 1
- с: сюда помещается вытолкнутый бит
- v: если произошло беззнаковое переполнение
- z: если результат равен 0
- n: если старший бит равен 0
##### rol rn
- rn = rn + rn + c
- c: если было беззнаковое переполнение
- v: не меняется
- z: если результат равен 0
- n: если старший бит равен 1
##### move rn, rm
- rm = rn
- c:  0
- v: 0 
- z: если результат равен 0
- n:если старший бит 1
