##### push rn
- помещает значение из регистра 
- стек указатель уменьшается на 1
##### pop rn
- помещает последний байт из стека в регистр
- стек указатель увеличивается на 1
##### pushall
- помещает регистры с 3 по 0 в стек
- смещает стек указатель
##### popall
- помещает значения из стека в регистры с 0 по 3
##### addsp
- изменяет указатель стека
##### lsda rn, offset
- загружает в регистр rn указатель на стек + offset
##### addsp const
- прибавляет к указателю на стек const
##### setsp const
- устанавливает указатель на стек на заданное значение
