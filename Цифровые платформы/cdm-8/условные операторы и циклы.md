### условные операторы:
```cdm-8
if 
	tst r0
is condidion
	neg r0
fi
```
### циклы:
```cdm-8
while 
	tst r1
stays condition
	add r0, r2
	dec r1
wend
```

```cdm-8
do
	#your code
until condition
```
##### break n
- останавливает выполнение n вложенных циклов
##### continue
- начинает новую итерацию с n вложенных циклов 
##### conditions:
- eq/z - z = 1
	- результат вычисления равен нулю 
- ne/nz - z = 0 
	- результат не равен нулю
- lo/cc - с = 1
	- переполнение при беззнаковой операции(255 + 1 = 0)
- hs/cs - с = 0
	- результат не вызвал переполнения
- mi - n = 1
	- после вычислений старший бит равен 1
- pl - n = 0
	- после вычислений старший бит равен 0
- vs - v = 1
	- переполнение при знаковой операции(127 + 1 = -128) 
- vc - v = 0
	- результат не вызвал переполнения 
- hi - c = 1 и z = 0
	- при сравнении 2 чисел A > B
- ls - c = 0 и z = 1
	- при сравнении 2 беззнаковых чисел A $\leq$ B
- ge - n == v
	- при сравнении 2 знаковых чисел A $\geq$ B
- lt - n != v
	- при сравнении 2 знаковых чисел A < B
- gt - $\neg$z && (n == v)
	- при сравнении 2 знаковых чисел A > B
- le - z || (n  != v)
	- при сравнении 2 знаковых чисел $A \leq B$
