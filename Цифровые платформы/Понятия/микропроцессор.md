#### микропроцессор - это интегральная микросхема, которая содержит все функции цп пк на одном микрочипе. он принимает входные данные, обрабатывает их и выдаёт результат
- задачи
	- выполнение кода
	- обработка данных
	- работа с озу
	- работа с другими компонентами
- основные элементы
	- арифметико-логическое устройство
	- устройство управления
	- регистры
	- шины - обеспечивает управление передачей данных
