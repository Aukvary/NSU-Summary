#### обратное отображение  - отображение $g: B \to A$, которое при композиции с отображением $f: A \to B$ образует диагональ => $f \circ g = id_A$ $g \circ f = id_A$ 

##### свойства:
- если для некоторого отображения $f: A \to B$ существует обратное отображение, то $f$ [[сюръективное отображение|сюръективно]] и [[всюду определённое отображение|всюду определено]] 
- **доказательство:**
	- пусть $f$ не сюръективно => $\exists b \in B: b \notin rng(f)$ 
	- по определению мы имеем $id_B = (g \circ f)(b) = f(g(b)) = b$ => если a = g(b), то $(a, b) \in f$ => противоречее
	- всюду определённость доказывается аналогично 
- если для некоторого отображения $f: A \to B$ существуе обратное отобажение, то $f$ [[инъективное отображение|инъективно]] 
- **доказательство**
	- пусть $f$ не инъективно => $\exists a_1, a_2 \in A\ \exists b \in B: a_1 \neq a_2\ \&\ f(a_1) = f(a_2) = b$
	- по определению $f \circ g = id_A$ => $\forall a \in A: (f \circ g)(a) = a$ 
	- $a_1 = (f \circ g)(a_1) = g(f(a_1)) = g(b) = g(f(a_2)) = (f \circ g)(a_2) = a_2$ => противоречее
- если для некоторого отображения $f: A \to B$ существует обратное отображение, то $f$ оно [[биектиное отображение|биективно]] 
- если для некоторого отображения $f: A \to B$ существует обратное отображение, то оно единственное
- **доказательство:**
	- пусть $\exists g_1, g_2: g_1: B \to A\ \&\ g_2:B \to A$ - 2 обратных отображения
	- $\exists b \in B: g_1(b) = a_1 \neq g_2(b) = a_2$ 
	- тк f сюръективно, то существует прообраз a при отображении f
		- $a = id_A(a) = (f \circ g_1)(a) = g_1(f(a)) = g_1(b) = a_1$
		- $a = id_A(a) = (f \circ g_2)(a) = g_2(f(a)) = g_2(b) = a_2$
	- $a_1 = a_2$ - противоречее
