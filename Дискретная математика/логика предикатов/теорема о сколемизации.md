#### для всякого предложения $\varphi$ справедливо: $\varphi$ - выполнимо $\Leftrightarrow Sk(\varphi)$ - выполнима

##### доказательство
- введём индукцию по числу $n$ кванторов $\exists$ в $\varphi$ уже имеющей вид [[пренексная нормальная форма(пнф)|пнф]] 
- если  n = 0, то $Sk(\varphi) = \varphi$ - то доказывать нечего 
- шаг индукции: сначала пусть $\forall x_1, ..., \forall x_n \psi(\bar x, f(\bar x))$ - выполнима
- тогда очевидно, что $\forall x_1, ..., \forall x_n \exists y\psi(\bar x, y)$ так же выполнима, при проверке полагаем $y = f(\bar x)$
- в другую сторону: если $\forall x_1, ..., \forall x_n \exists y\psi(\bar x, y)$ выполнена в структуре $\mathcal{M}$, то для всякого кортежа $\bar a \in M$(его элементы - значения переменных $x_i$) найдётся хотя бы один элемент $b$(значение $y$), для которого $\mathcal{M} \models \psi(\bar a, b)$ используя аксиому выбора, берём в качестве интерпретации символа $f$ операцию на $M$, сопоставляющую $\bar a$ ровно один соответствующий $b$  => $Sk(\varphi)$ выполнима
