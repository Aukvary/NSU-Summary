#### противоречивое множество формул - [[множество]] противоречиво, если для формул $\varphi_1, ..., \varphi_n \in \Phi$ [[секвенция(последовательность)|секвенция]] $\varphi_1, ..., \varphi_n \vdash \bot$ выводима 

##### свойство
- если множество $\Phi$ противоречиво, то оно не имеет модели 
- **доказательство:**
	- от противного, пусть $\Phi$ имеет модель $\mathcal{M}$ то есть при некоторой интерпретации $\gamma$ верно $\mathcal{M} \models \Phi[\gamma]$
	- тк $\Phi$ противоречиво, то найдутся $\varphi_1, ..., \varphi_n \in \Phi$ для которых секвенция $\varphi_1, ..., \varphi_n \vdash \bot$ доказуема 
	- по [[теорема о корректности логики предикатов|теореме корректности]] она [[формула предикатов тавтология(тождественная истина)|тождественная истина]], то есть в $\mathcal{M}$ при интерпретации $\gamma$, но тогда верно $\mathcal{M} \models \bot$ - противоречие
