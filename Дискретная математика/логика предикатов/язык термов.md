#### язык термов - множество слов [[алфавит логики предикатов|алфавита]] $A_{FOL}$ по индукции

##### индукция
- если $x \in V$ - предметная переменная => $x \in T(\sigma)$ - терм
- если $f^n \in F$ - функциональный символ, $t_1, ..., t_n \in T(\sigma)$ - терма, то слово $f(t_1, ..., t_n) \in T(\sigma)$ - тоже терм

##### обозначения
- слова из $T(\sigma)$ называются термами сигнатуры $\sigma$ 

##### глубина терма
- если $t = x \in V$ - предметная переменная => $d(t) = 0$
- если $t = f(t_1, ..., t_n) \in T(\sigma)$, где $f^n \in F$  => $d(t) = max(d(t_i)| 1 \leq i \leq n) + 1$ 
##### множество свободных переменных терма
- если $t = x \in V$ - предметная переменная => $V(t) = \{x\}$ 
- если $t = f(t_1, ..., t_n) \in T(\sigma)$, где $f^n \in F$  => $d(t) = V(t) = \cup\{V(t_i)|1 \leq u \leq n\}$
