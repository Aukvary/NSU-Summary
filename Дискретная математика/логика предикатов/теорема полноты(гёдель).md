#### пусть $\Phi$ - множество формул, $\psi$ - формула, если $\Phi \models \psi$, то $\Phi \vdash \psi$, то есть $\psi$ выводима из $\Phi$ 

##### доказательство
- пусть $\Phi \not\vdash$, тогда множество $\Phi \cup \{\neg \psi\}$ противоречиво 
- значит $\triangleright \varphi_1, ..., \varphi_n, \neg \psi \vdash \bot$ => $\triangleright \varphi_1, ..., \varphi_n \vdash \psi$ - противоречие 
- тогда по [[теорема о существовании модели(Гёдель)|теореме о существовании модели]] при подходящей интерпретации $\gamma$ имеем $\mathcal{M} \Phi \cup \{\neg \psi\}[\gamma]$ - это противоречит условию $\Phi \models \psi$ 

##### следствие
- если секвенция $s = \varphi_1, ..., \varphi_2 \vdash \varphi_0$ тождественная истина, то $s$ выводима
- **доказательство:**
	- если $s$ тождественная истина, то $\{\varphi_1, ... \varphi_n\} \models \varphi_0$ 
	- по теореме полноты $\varphi_0$ выводима из $\{\varphi_1, ..., \varphi_n\}$ 
	- тогда $\triangleright \varphi_{i_1}, ..., \varphi_{i_k} \vdash \varphi_0$ следовательно $\triangleright \varphi_{1}, ..., \varphi_{n} \vdash \varphi_0$
