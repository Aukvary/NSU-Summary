#### пусть $\varphi: M \to N$ - [[эпиморфизим структур|эпиморфизм]] [[структура алгебры|алгебр]] $\Rightarrow \exists \psi: M/\sim_{ker(\varphi)} \overset{\simeq}{\to}$, что $\psi$ - [[изоморфизм структур|изоморфизм]] и $\varphi = \nu_{ker(\varphi)} \circ \psi$  

##### доказательство
- определим отображение $\psi$ следующим образом 
- пусть $[a]_{\sim_{ker(\varphi)}}$ - некоторый [[класс эквивалентности]] из $M/\sim_{ker(\varphi)}$ 
- заметим, что $\forall a^\prime \in M [a]_{\sim_{ker(\varphi)}} = [a^\prime]_{\sim_{ker(\varphi)}}: a \sim_{ker(\varphi)} a^\prime$ 
- => $\varphi(a) = \varphi(a^\prime)$ => значение $\varphi(a)$ для класса $[a]_{\sim_{ker(\varphi)}}$ определено однозначно
- положим $g([a]_{\sim_{ker(\varphi)}}) = \varphi(a)$ => по определению $\nu_{ker(\varphi)} \circ \psi(a) = \varphi(a)$ 
- [[сюръективное отображение|сюръективность]] $\psi$ очевидна 
- [[инъективное отображение|инъективность]] - если $\psi([a]_{\sim_{ker(\varphi)}})$, то по определению $g$ имеем $\forall a, b \in M: \varphi(a) = \varphi(b)$ =>$a  \sim_{ker(\varphi)} b$ => $[a]_{\sim_{ker(\varphi)}} = [b]_{\sim_{ker(\varphi)}}$ => [[отображение(функция)|отображение]] является [[биектиное отображение|биективным]] 
- пусть $f^n \in \sigma$ - функциональный символ и $[a_1]_{\sim_{ker(\varphi)}}, .., [a_n]_{\sim_{ker(\varphi)}} \in M/\sim_{ker(\varphi)}$ 
- => $\psi(f^{M/\sim_{ker(\varphi)}}([a_1]_{\sim_{ker(\varphi)}}, .., [a_n]_{\sim_{ker(\varphi)}})) = \varphi(b)$, где $b \in X = [f^{M/\sim_{ker(\varphi)}}([a_1]_{\sim_{ker(\varphi)}}, .., [a_n]_{\sim_{ker(\varphi)}})]_{\sim_{ker(\varphi)}}$ 
- так как $f^M(a_1, ..., a_n) \in X$ => $\varphi(b) = \varphi(f^M(a_1, ..., a_n)) = f^N(\varphi(a_1), ..., \varphi(a_n)) = f^N(\psi([a_1]_{\sim_{ker(\varphi)}}), ..., [a_n]_{\sim_{ker(\varphi)}})$
