#### если множество формул $\Phi$ не имеет [[модель|модели]], то у него конечное подмножество $\Phi_0 \subset \Phi$ 

##### доказательство
- по [[теорема о существовании модели(Гёдель)|теореме о существовании модели]], раз $\Phi$ невыполнимо, то оно противоречиво, то есть $\Phi \vdash \bot$ 
- по определению тогда для подходящих $\varphi_1, ..., \varphi_n \in \Phi$ выводима $\varphi_1, ..., \varphi_n \vdash \bot$ 
- тем самым конечное множество $\Phi = \{\varphi_1, ..., \varphi_n\}$ - оно невыполнимо по [[теорема о корректности логики предикатов|теореме корректности]]
