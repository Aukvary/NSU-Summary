#### декартово произведение - [[структура(алгебраическая система)|структура]], определяемая как $(N, \sigma)$, где $N = M_1 \times ... \times M_n$, где $M_i$ это носитель структуры $\mathcal{M}_i$

##### интерпретация 
- пусть $f^k$- функциональный символ, $\bar a_1, ..., \bar a_k$ - последовательность [[кортеж|кортежей]](элементов носителя N)
- пусть $\bar a_i = (a^1_i, ..., a^n_i), a^i_j \in M_i$ тогда 
	- $f^\mathcal{N}(\bar a_1, ..., \bar a_k) = (f^{\mathcal{M}_1}(a_1^1, ..., a_k^1), f^{\mathcal{M}_2}(a_1^2, ..., a_k^2), ..., f^{\mathcal{M}_n}(a_1^n, ..., a_k^n))$ 
	- $(\bar a_1, ..., \bar a_k) \in p^\mathcal{N} \Leftrightarrow (a_1^1, ..., a_k^1) \in p^{\mathcal{M}_1} \land (a_1^2, ..., a_k^2) \in p^{\mathcal{M}_2} \land (a_1^n, ..., a_k^n) \in p^{\mathcal{M}_n}$
