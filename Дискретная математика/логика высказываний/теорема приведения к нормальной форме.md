#### $\forall \varphi: \exists \varphi^\prime$ в [[конъюктианя нормальная форма(кнф)|кнф]] $\land \varphi \sim \varphi^\prime$ 
#### $\forall \varphi: \exists \varphi^\prime$ в [[дизъюнктивная нормальная форма(днф)|днф]] $\land \varphi \sim \varphi^\prime$  

##### доказательство
- по [[Дискретная математика/логика высказываний/формула|леммам]] мы можем утверждать, что формула $\varphi$ имеет [[плотное отрицание]] и не содержит $\to$, теперь докажем теорему индукцией по глубине $\varphi$ 
- база индукции: если $\varphi$ атомарна, то $\varphi$ - [[литерал]] => кнф
- шаг индукции: пусть $d(\varphi) = n + 1$ и утверждение теоремы верно для всех формул глубины $\leq n$ 
- рассмотрим все возможные варианты построения $\varphi$
	- если $\varphi$ начинается с $\neg$, то $\varphi$ - [[литерал]], тк имеет [[плотное отрицание]]  
	- если $\varphi$ начинается со скобки, то у нас 2 случая(тк нет $\to$)
		- $\varphi = (\psi_1 \lor \psi_2)$ 
		- $\varphi = (\psi_1 \land \psi_2)$ 
	- рассмотрим случай $\varphi$ = $(\psi_1 \land \psi_2)$, тогда по предположению индукции для формул $\psi_i \exists \psi_j^\prime$ в кнф, что $\psi_i \sim \psi_j^\prime$ тогда по [[теорема о замене(семантическая версия)|по теореме о замене]]: $\varphi \sim (\psi_1^\prime \land \psi_2^\prime)$ - формула в кнф 
	- теперь рассмотрим $\varphi = (\psi_1 \lor \psi_2)$. по индукции существуют такие $\psi_j^\prime$ в кнф, что $\psi_i \sim \psi_j^\prime$ => $\varphi \sim (\psi_1^\prime \lor \psi^\prime_2)$
	- докажем утверждение теоремы индукцией по числу $m$ символов $\land$ 
	- если $m = 0$ => $(\psi_1^\prime \lor \psi_2^\prime)$ является [[элементарная конъюнкция, дизъюнкция|элементарной дизъюнкцией]] => кнф
	- пусть утверждение верно для $\forall k \leq m$, покажем, что оно верно для m + 1
	- рассмотрим случай, когда символ $\land$ встречается, например, в формуле $\psi_2^\prime$ => $\psi_2^\prime = \chi_1 \land \chi_2$ => по [[семантическая эквивалентность|дистрибутивности]] $\psi_1^\prime \lor \psi_2^\prime = \psi_1^\prime \lor (\chi_1 \land \chi_2) \sim (\psi_1^\prime \lor \chi_1) \land (\psi_1^\prime \lor \chi_2)$ 
	- формулы $\psi_1^\prime \lor \chi_1$ и $\psi_1^\prime \lor \chi_2$ имеют на 1 символ $\land$ меньше => по индукции существуют формы в кнф $\psi^{\prime\prime}_1 \sim \psi_1^\prime \lor \chi_1$ и $\psi_2^{\prime\prime} \sim \psi_1^\prime \lor \chi_2$ => $\varphi \sim (\psi_1^{\prime\prime} \land \psi_2^{\prime\prime})$ - искомая формула в кнф  
- приведение к [[дизъюнктивная нормальная форма(днф)|днф]] доказывается аналогично
