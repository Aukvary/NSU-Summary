#### полином Жегалкина - xor комбинация элементарных конъюнкций и константы 1(опционально) 

##### обозначения
- $\oplus$ - xor
- $ab$ - $a \land b$ 

##### эквивалентности
- $x\land y = x \oplus y$
- $\neg x = x \oplus 1$ 
- $x \oplus y = y \oplus x$ - коммутативность 
- $(x \oplus y) \oplus z = x \oplus (y \oplus z)$ - ассоциативность
- $x \oplus x = 0$
- $x \oplus 0 = x$ 
- $x \land (y \oplus z)  = (x \land y) \oplus (x \land z)$ - дистрибутивность $\land$ относительно $\oplus$
