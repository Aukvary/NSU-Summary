#### формула - [[слово]] в [[алфавит логики высказываний| алфавите логике высказываний ]]
- $\top, \bot, \forall i \in w: v_i$ - атомные(элементарные) формулы
- если $a, b$ формулы, тогда следующие слова тоже являются формулами:
	- $a \land b$
	- $a \lor b$
	- $a \rightarrow b$
	- $\neg a$ 

##### типы формул
- [[формула тавтология]]
- [[выполнимая формула]]
- [[невыполнимая формула]]

##### множество переменных
- $V(\varphi)$ - множество переменных 
- $V(\top) = V(\bot) = \varnothing$ 
- $\forall v_i \in V: V(v_i) = \{v_i\}$ 
- $V(\neg \varphi) = V(\varphi)$ 
- $V(\varphi \bullet \psi) = V(\varphi) \cup V(\psi), \bullet \in \{\land, \lor, \to\}$ 

##### глубина формулы
- $d(\varphi)$ - глубина формулы
- $d(\top) = d(\bot) = 0$ 
- $\forall v_i \in V: d(v_i) = 0$ 
- $V(\neg \varphi) = V(\varphi) + 1$ 
- $V(\varphi \bullet \psi) = max(d(\varphi), d(\psi)) + 1, \bullet \in \{\land, \lor, \to\}$ 

##### лемма
-  $\forall \varphi: \exists \varphi^\prime: \varphi \sim \varphi^\prime \land \to \notin \varphi$ 
- **доказательство:**
	- индукция по глубине $\varphi$ 
	- база индукции: $d(\varphi) = 0 \Rightarrow \varphi \in \{v, \top, \bot\}$ => утверждение очевидно 
	- шаг индукции: пусть $d(\varphi) = n + 1$, тогда имеет место один из вариантов
		- $\varphi = \neg \psi$
		- $\varphi = (\psi_1 \land \psi_2)$
		- $\varphi = (\psi_1 \lor \psi_2)$
		- $\varphi = (\psi_1 \to \psi_2)$ 
	- поскольку $d(\psi_i) \leq n$, то для формул $\psi_i \exists \psi_j^\prime \sim \psi_i: \to \notin \psi_j^\prime$ 
	- рассмотрим все случаи
		- $\varphi^\prime = \neg \psi_1^\prime \sim \neg \psi_1 = \varphi$ 
		- $\varphi^\prime = (\psi^\prime_1 \land \psi^\prime_2) \sim (\psi_1 \land \psi_2) = \varphi$ 
		- $\varphi^\prime = (\psi^\prime_1 \lor \psi^\prime_2) \sim (\psi_1 \lor \psi_2) = \varphi$ 
		- используя [[семантическая эквивалентность|эквивалентность]], [[лемма о подстановке в эквивалентные формулы|лемму о подстановке]] и [[теорема о замене(семантическая версия)|теорему о замене]] получаем $\varphi^\prime = (\neg \psi_1^\prime \lor \psi_2^\prime) \sim (\psi_1^\prime \to \psi_2^\prime) \sim (\psi_1 \to \psi_2) = \varphi$ 
##### лемма 
- $\forall \to \notin \varphi: \exists \varphi^\prime: \varphi \sim \varphi^\prime$, где $\varphi^\prime$ - [[плотное отрицание]] 
- **доказательство:**
	- индукция по глубине $\varphi$
	- база индукции: атомарные формулы => очевидно, тк $\neg$ нет
	- пусть $d(\varphi) = n + 1$, и для формул глубины $\leq n$ утверждение леммы выполняется 
	- рассмотрим варианты структуры $\varphi$ 
	- если $\varphi  = (\psi_1 \bullet \psi_2), \bullet \in \{\land, \lor\}$, то по предположению индукции для формул $\psi_1$ и $\psi_2$ $\exists \psi_1^\prime, \psi_2^\prime: \psi_1 \sim \psi_1^\prime \land \psi_2 \sim \psi_2^\prime$, где $\psi_1^\prime, \psi_2^\prime$ - [[плотное отрицание]] 
	- => $\varphi \sim (\psi_1^\prime \bullet \psi_2^\prime) = \varphi^\prime$ - искомая формула с плотным отрицанием 
	- теперь рассмотрим вариант $\varphi = \neg \psi$, где $d(\psi) > 0$, проверим $\psi$:
		- **вариант 1:** $\psi = \neg \chi$ => по предположению индукции $\chi \sim \chi^\prime$, где $\chi^\prime$ формула с тесными отрицанием 
		- тогда $\varphi \sim \neg\neg\chi \sim \chi \sim \chi^\prime = \varphi^\prime$ 
		- **вариант 2:** $\psi = (\chi_1 \land \chi_2)$ => $\varphi \sim \neg(\chi_1 \land \chi_2) \sim (\neg \chi_1 \lor \neg \chi_2)$ 
		- по индукции $\exists \neg \chi_i \sim \chi_i^\prime$, что $\chi_i^\prime$ плотное отрицание 
		- => $\varphi \sim (\chi_1^\prime \lor \chi_2^\prime) = \varphi^\prime$
		- **вариант 3:** $\psi = (\chi_1 \land \chi_2)$ доказывается аналогично второму варианту
