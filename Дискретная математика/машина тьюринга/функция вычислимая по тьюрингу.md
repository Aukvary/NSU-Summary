#### функция вычислимая по тьюрингу - частично числовая функция $f: \mathbb{N}^n \to \mathbb{N}$, для которой существует [[машина тьюринга]] $M$ такая, что $\forall a_1, ..., a_n \in \mathbb{N}$ выполняется несколько условий

##### условия
- если $f(a_1, ..., a_n)$ определено, то $M$, начав работу с лентой, содержащей коды чисел $a_1, ..., a_n$, останавливается и на ленте остаётся число $b = f(a_1, ..., a_n)$ 
- если $f(a_1, ..., a_n)$ не определено, то $M$ не останавливается 
- может быть представлена в виде [[множество лямбда-термов|лямбда исчислении]] 
- может быть представлена в виде [[частично рекурсивная функция|частично рекурсивной функции]]
