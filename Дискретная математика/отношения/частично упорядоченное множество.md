#### частично упорядоченное множество - упорядоченная пара (А, $\leq$), где А - множество, а $\leq$ это [[отношение частичного порядка]] 
- наименьший элемент
	- $\forall a \in A\ (a_0 \leq a)$ 
- минимальный элемент
	- $\forall a \in A\ (a \leq a_0$ => $a = a_0)$ 
- наибольший элемент
	- $\forall a \in A\ (a \leq a_0)$
- максимальный элемент
	- $\forall a \in A \ (a_0 \leq a$ => $a_0 = a)$
