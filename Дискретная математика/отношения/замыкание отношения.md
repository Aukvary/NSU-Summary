#### замыкание отношения - это такое [[бинарные отношения|отношение]] $r*$, которое содержит исходное отношение $r \subseteq r*$, обладает свойством $P$ и является наименьшим 

##### способы получения для множества A и отношения r:
- [[рефлексивное замыкание]] 
- [[симметричное замыкание]]
- [[транзитивное замыкание]]

##### свойства:
- если $\exists r^*$ => оно единственное и совпадает с множеством $cl_P = \cap\{r^{\prime}|r \subseteq r^{\prime} \land r^{\prime} \in P\}$ 
- **доказательство:**
	- пусть существует другое замыкание $r^{**}$ для $r$ относительно P
	- => $r \subseteq r^*$ => по определению замыкания $r^{**} \subseteq r^*$ 
	- с другой стороны используя определение для $r* \subseteq r^{**}$ => $r^* = r^{**}$ 
	- теперь предположим, что $r^\prime$ существует 
	- докажем, что $cl_P(r) \subseteq r^*$ => верно тк $r* \in \cap\{r^{\prime}|r \subseteq r^{\prime} \land r^{\prime} \in P\}$ 
	- теперь докажем, что $r^* \in cl_P(r)$ - верно, тк $r^*$ - минимальный элемент
