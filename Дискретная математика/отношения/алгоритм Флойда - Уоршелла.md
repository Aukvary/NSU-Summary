#### алгоритм Флойда - Уоршелла - для отношение $r$ на конечном множестве $A$ и его матрица смежности $M(r)$ его транзитивное замыкание может быть вычислено следующим образом:
```
W = M(r)

for k = 1 to n
	for i = 1 to n
		for j = 1 to n
			W[i][j] = W[i][j] || (W[i][k] && W[k][j])
```
