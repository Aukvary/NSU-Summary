##### свойтсва
- если существует наибольший/наименьший элемент, то он единственный
- **доказательство:** ^dcac3e
	- $\leq$ - частичный порядок на множестве $А$, $X \subseteq A$. 
	- $a_1, a_2 \in A$ - наименьшие элементы => по определния $a_1 \leq a_2$ и $a_2 \leq a_1$ 
	- тк $\leq$ [[антисимметричные отношения|антиметрично]], то $a_1 = a_2$ 

##### лемма
- если $\exists a = max(X) \Rightarrow sup_A(X) = a$ 
- если $\exists b = min(X) \Rightarrow inf_A(X) = b$ 
- **доказательство:**
	- пусть $a = max(X)$  => $\forall c \in X: c \leq a$ => $a \in (X \uparrow A)$ 
	- пусть $b = sup_A(X)$ и $a \neq b$ => $b \lt a$ 
	- тк b - верхняя граница, то $a \leq b$
	- второе доказывается аналогично
