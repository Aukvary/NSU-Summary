#### линейно упорядоченное множество - [[частично упорядоченное множество(ЧУМ)]], у которого [[отношение линейного порядка]]  

##### свойства
- любое лум является [[решётка|решётка]] 
- **доказательство:**
	- пусть $(A, \leq)$ - лум
	- $\forall a, b \in A: a \leq b \lor b \leq a$ тк $\leq$ - отношение частичного порядка
	- => $min(\{a, b\}) = a \land max(\{a, b\}) = b$ во втором случае наоборот => для $X := {a, b} \exists min\ max$ => по лемме о[[частично упорядоченное множество(ЧУМ)#лемма| лемме]] sup и inf существуют
