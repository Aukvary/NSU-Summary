#### множество всех подмножеств - всех подмножества, которые содержаться в множестве
- $P(A) := \{X | X \subseteq A\}$  

##### теорема:
- $A_n := \{a_1, ..., a_n\}$ => $P(A_n)$ состоит из $2^n$ элементов
- **доказательтво:**
	- возьмём индукцию по n
	- n = 0 => $P(\varnothing) = \{\varnothing\}$ состоит из 1 элемента
	- предположим, что утверждение доказано для n - 1, то есть $P(A_{n - 1})$ состоит из $2^{n - 1}$ элементов
	- тогда произвольное множество $A_n = A_{n-1} \cup \{a_n\}$ 
	- тк в $P(A_{n - 1})$ элементов, и каждое подмножество из $P(A_{n - 1}) пораждает ровно 2 подмножества в $P(A_n)$: содержащее $a_n$ и не содержащее его,, то всего в $P(A_n)$ будет 2 * $2^{n - 1} = 2^n$ элементов
