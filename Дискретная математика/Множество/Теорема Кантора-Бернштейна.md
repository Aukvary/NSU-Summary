#### Теорема Кантора-Бернштейна - $A \preceq B, B \preceq A$ <=> $A \approx B$ 

##### доказательство
- по условию $\exists f: A \overset{1:1}{\longrightarrow} B \land \exists g: B \overset{1:1}{\longrightarrow} A$  => нам необходимо построить биекцию $f: A \overset{1-1}{\twoheadrightarrow} B$
- построим последовательность множеств $A_i \subseteq A, i \in w$ 
- в начале $A_0 = A$ и $A_1 = g(B)$ 
- другие члены последовательности: $\forall n > 1: A_{n + 2} = (f \circ g)(A_n)$ 
- нам нужно показать, что $\forall n: A_{n+1} \subseteq A_n$ 
- база индукции: $A_1 = g(B) \subseteq A_0 = A$
- шаг индукции: $\forall i < n: A_{i + 1} \subseteq A_i$ 
- пусть $a \in A_{n + 1} = (f \circ g)(A_{n - 1})$ => $\exists b \in A_{n  - 1} \subseteq A_{n - 2}: a = g(f(b))$ =>$a \in g(f(A_{n - 2})) = A$ 
- теперь рассмотрим $D = \cap_{n \in w}A_n$ и определим $\forall n \in w: M_i = A_i \ A_{i + 1}$ => $f \circ g: M_i \rightarrow M_{i +2}$ 
- теперь определим отображение $h: A \rightarrow A$:
	- если $a \in \cup_{n \in w} M_{2n}$
	- если $a \in (\cup_{n \in w} M_{2n + 1}) \cup D$ 
- образ отображения h: область значений = $(\cup_{n \in w}M_{n}) \cup D = A_1$ => $h: A \rightarrow A_1$ 
- тк $g: B \rightarrow A_1$ => $h \circ g^{-1}: A \rightarrow B$ - биекция
