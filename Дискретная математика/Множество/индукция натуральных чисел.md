#### индукция натуральных чисел:
- $0 := \varnothing$ 
- $n + 1 = n \cup \{n\}$ 
##### пример:
- 0 = ∅ (пустое множество)
- 1 = 0 ∪ {0} = ∅ ∪ {∅} = {∅} = {0}
- 2 = 1 ∪ {1} = {0} ∪ {1} = {0, 1}
- 3 = 2 ∪ {2} = {0, 1} ∪ {2} = {0, 1, 2}
- …
- n = {0, 1, 2, ..., n-1}
