#### индукция натуральных чисел:
- $0 := \varnothing$ 
- $n + 1 = n \cup \{n\}$ 

##### обозначение
- $w$ - множество всех натуральный чисел 

##### пример:
- 0 = ∅ (пустое множество)
- 1 = 0 ∪ {0} = ∅ ∪ {∅} = {∅} = {0}
- 2 = 1 ∪ {1} = {0} ∪ {1} = {0, 1}
- 3 = 2 ∪ {2} = {0, 1} ∪ {2} = {0, 1, 2}
- …
- n = {0, 1, 2, ..., n-1}
##### свойства
- P([[индукция натуральных чисел|w]]) - [[счётные-несчётные множества|не счётное]] по [[Дискретная математика/Множество/теорема Кантора|теореме Кантора]] и имеет мощность [[континуум|континуума]]
