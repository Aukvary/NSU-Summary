#### теорема Кантора - $\forall A: A \prec P(A)\ \land A \cancel{\approx} P(A)$ 

##### доказательство 
- [[инъективное отображение|инъективное]]  отображение $a \overset{1:1}{\longrightarrow} \{a\}$ $A \overset{1:1}{\longrightarrow} P(A)$ показывает, что $A \preceq P(A)$
- пусть $A \approx P(A)$ => $\exists f: A \overset{1-1}{\twoheadrightarrow} P(A)$ 
- далее определим множество $B = \{a|a \notin f(a)\}$ и его прообраз $a = f^{-1}(B)$, то есть $f(a) = B$ 
- проверим $a \in B$
	- пусть $a \in B = f(a)$ => $a \notin B$, по определению B - противоречие
	- пусть $a \notin B = f(a)$ => по определению B $a \in B$ - противоречие 
- => такой функции не существует
