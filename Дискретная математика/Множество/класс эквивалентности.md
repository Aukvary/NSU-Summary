#### класс эквивалентности - это множество элементов, где отношение между элементами [[рефлексивные отношения|рефлексивно]], [[симметричные отношения|симметрично]] и [[транзитивная отношения|транзитивно]] 

##### обозначения:
- $[a]_{\sim}$ := $\{b|b \in A \land a \sim b\}$ - класс эквивалентности элемента а по эквивалентности r

##### свойства:
- $a \in [a]_\sim$ 
- **доказательство:**
	- следует из эквивалентности
- $[a_1] \cap [a_2] \neq \varnothing$ => $[a_1] = [a_2]$ 
- **доказательство:**
	- пусть $b \in [a_1]_\sim \cap [a_2]_\sim$ => по определению класса эквивалентности $a_1 \sim b$ и $a_1 \sim b$, тк отношение [[симметричные отношения|симметрично]] и [[транзитивная отношения|транзитивно]] => $a_1 \sim a_2$ 
	- пусть $b \in [a_1]_\sim$, тк $a_1 \sim a_2$ => $b \sim a_2$ => $b \in [a]_\sim$ по определению класса эквивалентности => $[a_1]_\sim \subseteq [a_2]_\sim$ аналогично если поменять $a_1$ и $a_2$ местами => $[a_1] = [a_2]$ 
- $A = \cup \{[a] | a \in A\}$ 
- **доказательство:**
	- выводится из 1

##### теорема:
- пусть $I$ - множество эквивалентностей на множестве A => отношение $r := \cap I$ является эквиваленостью
- **доказательства:**
	- рефлексивность: $\forall \sim: id_A \subseteq \sim$ => $id_A \subseteq \cap I$ 
	- симметричность: пусть $(a, b) \in r$ => $\forall \sim \in I: (a, b) \in \sim$, тк $\sim$ симметрично => $(b, a) \in \sim$ => $(b, a) \in r$ 
	- транзитивность: пусть $(a, b) \in r$ и $(b, c) \in r$ => $\forall \sim \in I: a \sim b \sim c$ => $a \sim c$ => $(a, c) \in r$
