#### разбиение - такое подмножество  [[множество всех подмножеств(булеан)|множества всех подмножеств]] X множества А, где подмножества не пересекаются 

##### свойства:
- $\varnothing \notin X$ 
- $\forall a, b \in X: a \cap b \neq \varnothing$ <=> $a = b$ 
- $A = \cup X$ 

##### лемма:
- пусть $X \subseteq P(A)$ - разбиение множества А
- определим бинарное отношение $\sim$ - $a \sim b$ <=> $\exists x \in X: a \in x \land b \in x$  
- $\sim$ - [[отношение эквивалентности]] и $X = \{[a]_\sim|x \in A\}$ 
- **доказательство:**
	- [[симметричные отношения|симметричность]] следует из определения
	- так как $A = \cup X$ => a попадает в какой-то элемент разбиения $a \in x \in X$ => отношение [[рефлексивные отношения|рефлексивно]]
	- пусть $a \sim b$ и $b \sim c$ => $\exists x, y \in X: a, b \in x \land b, c \in y$ => $b \in x \cap y \neq \varnothing$ =>x = y => $a, c \in x$ => $a \sim c$ 
	- пусть $x \in X => $x \neq \varnothing$ => $\exists a \in x$ 
	- => $\forall b: b \sim a$ <=>$b \in x$, потому что если $a \sim b$, то для некоторого $y \in X$: $a, b \in y$ 
	- =>$[a]_\sim \subseteq x$ 
	- обратно, если некоторый $b \in X$, то по определению $\sim$, $b \sim a$ => $b \in [a]_\sim$ => $x = [a]_\sim$ 
	- если $[a]_\sim$ - некоторый класс эквивалентности, то в силу $A = \cup X$, имеем $a \in x$ для некоторого $x \in X$, далее всё аналгочно первой чити
