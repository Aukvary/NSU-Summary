#### конечное множество - множество, которое равномощно некоторому натуральному числу [[индукция натуральных чисел|n]] 

##### обозначения
- n - мощность

##### теорема
- A конечно если $\forall B \subset A \land B \neq A: A \not\approx B$    
- **доказательство:**
	- пусть А конечно и $A = n \in w$, а также существует некоторая биекция $f: n \overset{1-1}{\twoheadrightarrow} B$
	- возьмём $|B| = m < n$ - так же натуральное число
	- теперь докажем невозможность по индукции
	- если n = 1 => m = 0 - пустое множество но оно должно содержать $f(0)$ - противоречие
	- шаг индукции: $n = \{0, 1, ..., n - 1\} > 1$ 
	- теперь возьмём возьмём ограничение $f^\prime$ отображения $f$ на множестве $n - 1 = \{0, 1, ..., n - 2\}$ => биекция $f^\prime: n - 1 \overset{1-1}{\twoheadrightarrow} m -1$ 
	- под предположению индукции эта биекция возможна при $n - 1 = m - 1$ => противоречие  
- **доказательство:**
	- пусть $\forall B \subset A: A \not\approx B$, докажем, что $\exists n: A \preceq n$ 
	- если такого $n$ нет, то $\forall n: n \preceq A$ => $\exists f_n: n \rightarrow A$ 
	- мы можем выбрать эти отображения так, что есть $n \leq m \Rightarrow f_m \subseteq f_n$ 
	- зададим отображение $f_n$ и построим $f_{n + 1}$ 
	- заметим, что $cod(f_{n + 1}) \not\subseteq cod(f_n)$, тк $n + 1 \not\preceq n$ 
	- => $\exists a \in cod(f_{n + 1}) \backslash cod(f_n)$ => $f^\prime_{n + 1} = f_n \cup \{(n, a)\}$ остаётся инъективным 
	- беря $\cup\{f_n | n \in w\}$ => инъективное отображение $f: w \to A$ => $\exists A^\prime \subset A: A^\prime \approx A$
