#### декартовое произведение множеств 
- $A \times B := \{(a, b) | a \in A, b \in B\}$ 
##### декартова степень:
- если $A_1$ = $A_2$ =  $\cdots$ = $A_n$ = $A$, то их произведение выгладит как $A^n$ 
##### ассоциативность
- $A \times (B \times C) \neq (A \times B) \times C$ 
- **доказательство:**
	- $a \in A, b \in B, c \in C$
	- $A \times (B \times C) = (a, (b, c))$ 
	- $(A \times B) \times C = ((a, b), c)$ 
	- чтд
##### лемма 1:
- для любых множеств $A_1, ..., A_n$ $A_1 \times ... \times A_n = (A_1 \times ... \times A_{n-1}) \times A_n$ 
- **доказательтво:**
	- по определению декартова произведения:
	- $A_1 \times ... \times A_n$ = $\{(a_1, ..., a_n)|a_i \in A_i\}$ = $\{((a_1, ..., a_{n - 1}), a_n)|a_i \in A_i\}$ = $\{(\bar{a}, a_n)|a_n \in A_n, \bar{a} \in A_1 \times ... \times A_{n-1}\}$ 
##### лемма 2:
- $A \times B$ содержит len(A) * len(B) элементов
- **доказательство:**
	- пусть len(A) = n, а len(B) = m
	- тогда у нас есть n способ выбрать элемет из A и m способ выбать элемент из B для пары (a, b)
	- => $A \times B$ coдержит n * m элементов 
##### теорема:
- $A_1, ..., A_n$ - последовательность конечных множеств, где $A_i$ содержит содержит $k_i$ элементов => $A_1 \times ... \times A_n$ будет содержать $k_1 * ... * k_n$ 
- **доказательство:**
	- база индукции n = 1 - очевидно
	- для n = 2 - лемма 2
	- предположим, что $A_1 \times ... \times A_{n - 1}$ имеет $k_1 * ... * k_{n -1}$ элементов
	- по лемме 1 $X = $A_1 \times ... \times A_n = (A_1 \times ... \times A_{n - 1}) \times A_n$  
	- по лемме 2 X содержит $(k _1, ..., k_{n -1}) * k_n$
