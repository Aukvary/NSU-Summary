#### хроматическое число - минимальное количество цветов, необходимое для правильной [[правильная раскраска графа|раскраски графа]] $G$

##### обозначения
- $\chi(g)$
- $k$-раскрашиваемый - $\chi(G) \le k$

##### свойства
- хроматическое число графа триангуляции $G_T$ любого простого многоугольника равно 3
