#### хроматическое число - минимальное количество цветов, необходимое для правильной [[раскраска графа|раскраски графа]] $g$, обозначаемое как $\chi(g)$

##### знание этого числа позволяет определить минимальные ресурсы (например, количество аудиторий или временных интервалов), необходимые для выполнения всех задач без конфликтов

##### свойства
- хроматическое число графа триангуляции $G_T$ любого простого многоугольника равно 3
