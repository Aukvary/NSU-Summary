#### алгоритм
- добавить в граф новую вершину $s$ и ребра с нулевым весом до всех остальных [[вершина|вершин]]
- использовать [[алгоритм беллмана-форда|алгоритм беллмана-форда]] для вычисления кратчайших расстояний $h(v)$ от $s$
- изменить веса всех ребер на $w'(u, v) = w(u, v) + h(u) - h(v)$, чтобы сделать их неотрицательными
- для каждой вершины $u$ запустить [[алгоритм дейкстры|алгоритм дейкстры]] с новыми весами $w'$
- вычислить итоговые расстояния, вернувшись к исходным весам через формулу $d_{uv} = \delta'(u, v) + h(v) - h(u)$
- эффективен для [[разреженный граф|разреженных графов]]
