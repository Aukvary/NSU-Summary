#### число остовных деревьев полного графа $K_n$ с $n$ помеченными вершинами строго равно $\tau(K_n) = n^{n-2}$ 

##### доказательство
- установим биекцию между множеством всех остовных в полного графа на множестве вершин $\{1, 2, ..., n\}$ и множеством последовательностей (кодов прюфера) длины  $n-2$, элементами которых является целые числа от 1 до $n$ 
- **алгоритм построения кода по дереву**
	- на каждом шаге $i \in [1, n-2]$ находим в текущем дереве лист с наименьшим числовым номером
	- записываем номер единственной смежной с этим листом вершины в качестве очередного элемента $t_i$ кода  прюфера
	- удаляем лист и инцидентное ему ребро из дерева
- процесс останавливается, когда в дереве остаётся ровно 2 вершины
- полученный упорядоченный дочерний набор $(t_1, t_2, ..., t_n)$ является кодом прюфера для исходного дерева
- поскольку этот алгоритм однозначно обратим, существует взаимно-однозначное соответствие
- количество способов составить последовательность длины $n-2$ из $n$ возможных символов равно $n \times n \times ... \times n = n^{n-2}$ 
- следовательно, число различных остовных деревьев полного графа также равно $n^{n-2}$
