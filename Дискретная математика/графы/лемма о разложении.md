#### пусть $G = (V, E)$ - простой неориентированный граф, и в нём существует хотя бы одна пара несмежных вершин $v, w$, обозначим за $e$ теоретическое ребро между ними введём две вспомогательные графовые структуры $G + e$ -граф, полученный из $G$ путём прямого добавления ребра $e = \{v, w\}$ и $G /e$  - граф полученный стягиванием этих вершин, все кратные рёбра сливаются в обычные рёбра 
#### для любой пары несмежных вершин $v, w$ графа $G$ справедливо следующее $$\chi(G) = \min(\chi(G+e), \chi(G/e))$$

##### доказательство
- рассмотрим любую правильную вершинную раскраску исходного графа $G$ 
- для вершин $v, w$ возможны только два непересекающихся
	- **вершины имеют разные цвет:**
		- такая раскраска правильная для ситуации $G+e$, тк добавление ребра создаёт конфликт раскраски
		- количество цветов равно строго $\chi(G +e)$ 
	- **вершины имеют одинаковый цвет:**
		- такая раскраска правильная для ситуации $G/e$, где слитая вершина $x_{vw}$ принимает общий цвет
		- минимальное минимальное число для графа составляет $\chi(G/e)$
