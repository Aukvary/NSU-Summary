#### спектр графа - [[множество]] собственных значений в порядке возрастания 

##### обозначения
- $spec(G)$ 

##### свойства
- для [[ориентированный граф|неориентированного графа]] все $\lambda_i \in \mathbb{R}$, тк [[матрица]] $A$ симметрична
- **оценка собственных значений:** максимальное собственное значение $\lambda_{max}$ ограничено средней и максимальной степенями графа $$\frac{2|E|}{n} \leq \lambda_{max} \leq \Delta(G)$$
- **регулярные графы:** [[граф]] является [[регулярный граф|регулярным]] тогда и только тогда, когда вектор из одних единиц $e = (1, ..., 1)^T$ является его [[собственный вектор|собственным вектором]] с собственным значением $k$ 
- **спектр полного графа:** [[полный граф]] $K_n$ имеет спектр, состоящий из числа $n-1$ с кратностью 1 и числа -1 с кратностью $n-1$
