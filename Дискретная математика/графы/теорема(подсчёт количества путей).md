#### для любого [[граф|графа]] $G$ с множеством вершин $V = \{v_1, v_2, \cdots, v_n\}$, элемент $(i, j)$ [[матрица смежности(графы)|матрицы смежности]] $A^k$ представляет собой количество [[путь(неориентированный граф)|путей]] из вершины $v_i$ в вершину $v_j$ длины $k$ 

##### доказательство
- индукция по $k$ 
	- для $k$ = 1: $A(i, j) = 1$ - путь есть и его длина равна 1, $A(i, j) = 0$ - путь отсутствует 
	- пусть утверждение верно для некоторого $k \geq 1$, и рассмотрим количество путей длины  $k + 1$ из вершины $v_i$ в вершину $v_j$ 
	- любой путь длины $k + 1$ из $v_i$ в $v_j$ содержит путь длины $k$ из $v_i$ в некоторого соседа вершины $v_j$ 
	- если вершина $v_p \in N(v_j)$, то по индукции количество путей длины $k$ из $v_1$ в $v_p$ равно $A^k(i, p)$ $\Rightarrow$ общее число путей длины $k + 1$ из $v_i$ в $v_j$ равно $$\underset{v_p \in N(v_j)}\Sigma A^k(i, p) = \overset{n}{\underset{i=l}{\Sigma}}A^k(i, l)A(l, j) = A^{k + 1}(i, j)$$
