#### задача о картинной галерее - прикладная задача определения минимального числа охранников, необходимых для наблюдения за всеми точками внутри простого $n$-угольника

##### точка внутри многоугольника считается видимой охранником, если отрезок между ними не пересекает границы фигуры, в классической постановке охранники размещаются только в вершинах, задача является вычислительно сложной, поэтому для нее ищут приближенные решения

##### свойства
 - верхняя оценка количества охранников для произвольного n-угольника $floor(n / 3)$

##### шаги
- триангуляция многоугольника диагоналями 
	- количество треугольников равно $n-2$ для любого произвольного n-угольника
