#### формула эйлера - фундаментальное соотношение для [[связный граф|связного]] [[планарный граф|планарного графа]], связывающее количество вершин $n$, ребер $m$ и [[грань плоского графа|граней]] $f$: $n - m + f = 2$

##### доказательство
- **база индукции:**
	- пусть граф $G$ является [[остовное дерево|остовным деревом]] $T$
	- из свойств деревьев известно, что количество рёбер в нём строго равно $m = n -1$ 
	- поскольку в дереве нет циклов, оно не может ограничивать внутренние области, а значит любая его плоская укладка содержит ровно одну внешнюю грань
	- подставим значения $$n - m + f = n - (n - 1) + 1 = n - n + 1 + 1 = 2$$
	- чтд
- **индукционный шаг:** 
	- пусть для некоторого подграфа исходного графа равенство выполняется 
	- будем поочерёдно возвращать в граф по одному ребру, не входящих в остовное дерево
	- каждое такое добавляемое ребро соединяет две уже существующие вершины графа
	- поскольку граф связен, между этими вершинами в остове уже существовал простой путь, таким образом добавление ребра замыкает новый простой цикл
	- геометрически на плоскости этот цикл гарантированно разрезает одну из существующих граней на две независимые части
	- в результате это шага:
		- количество рёбер увеличивается на 1: $m^\prime = m + 1$
		- количество граней увеличивается на 2: $f^\prime = f + 1$
		- количество вершин не меняется: $n^\prime = n$
	- запишем новое соотношение $$n^\prime - m^\prime + f^\prime = n - (m + 1) + (f + 1) = n - m - 1 + f + 1 = n - m + f$$
	- $n-m+f=2 \Rightarrow n^\prime - m^\prime + f^\prime = 2$ 

##### следствие 1
- для любого простого связного планарного графа с количеством вершин $n \ge 3$ выполняется равенство $m \le 3n - 6$ 
- **доказательство:**
	- каждая грань $F$ простого графа должна быть ограничена некоторым замкнутым маршрутом
	- минимально возможный цикл в простом графе - это треугольник, поэтому степень любой грани удовлетворяет неравенству $\deg(F) \ge 3$ 
	- подсчитаем сумму степеней всех граней плоской укладки
	- поскольку каждое ребро графа граничит ровно с двумя гранями, сумма граней строго равна удвоенному числу рёбер $$2m = \sum^f_{i=1}\deg(F_i) \ge 3f \Rightarrow f \le \frac{2m}{3}$$
	- поставим полученную верхнюю оценку для количества граней $f$ в исходной формулу эйлера $$2 = n - m + f \le n - m + \frac{2m}{3} \Rightarrow 2 \le n - \frac{m}{3} \Rightarrow m \le 3n - 6$$

##### следствие 2
- в любом простом графе связном планарном графе гарантированно существует хотя бы одна вершина, степень которой не превосходит 5
- **доказательство:**
	- пусть в некотором простом планарном графе степени абсолютно всех вершин без исключения не меньше 6
	- используя [[теорема о рукопожатии|теорему о рукопожатии]], подсчитаем сумму степеней всех вершин графа $$2m = \sum_{v\in V} \deg(v) \ge \Rightarrow m \ge 3n$$
	- согласно **следствию 1**, для любого простого планарного графа должно выполнятся ограничение $m \le 3n - 6$ 
	- мы получили противоречие $3n \le m \le 3n - 6 \Rightarrow 0 \le -6$, следовательно предположение неверно 
##### следствие 3
- если простой связный граф является планарным и одновременно двудольным при $n \ge 3$, то для него выполняется более строгое ограничение на рёбра $m \le 2n -4$ 
- **доказательство:**
	- согласно [[теорема кёнинга|критерию кёнига]], двудольный граф принципиально не содержит простых циклов нечётной длины
	- это значит, что в нём отсутствуют треугольники, и минимальная длина простого цикла равна 4
	- повторим действия из 1 следствия $$2m = \sum^f_{i=1} \deg(F_i) \ge 4f \Rightarrow f \le \frac{m}{2}$$
	- подставим оценку в формулу эйлера $$2 = n - m + f \le n - m + \frac{m}{2} \Rightarrow 2 \le n - \frac{m}{2} \Rightarrow m \le 2n - 4$$

##### доказательство непланарности
- **полный граф $K_5$:** 
	- данный граф имеет $n = 5$ вершин и $m = 10$ рёбер
	- подставим в **следствие 1** $$3n - 6 = 3 \times 5 - 6 = 15 - 6 = 9$$
	- поскольку число рёбер больше максимума, граф не планарный 
- **полный двудольный граф $K_{3, 3}$:**
	- данный граф имеет $n = 6$ вершин и $m = 9$ рёбер 
	- подставим в **следствие 3** $$2n - 4 = 2\times 6 = 12 \times 4 =8$$
	- тк количество рёбер превышает максимум граф не планарный
