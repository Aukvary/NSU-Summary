#### по завершении работы алгоритма [[построение остовного дерева (BFS)|BFS]] для любой вершины $v \in V: v.d \geq \delta(s, v)$ 

##### доказательство
- **база индукции:** в момент инициализации $s.d = 0 = \delta(s, s)$  и $\infty \geq \delta(s, v)$ для всех $v \neq s$ 
- **шаг индукции:**
	- пусть белая вершина $v$ обнаруживается при просмотре соседей вершины $u$
	- в этот момент ей присваивается значение $v.d = u.d + 1$ по индуктивному предположению для вершины $u$ уже выполнилось $u.d \ge \delta(s, u)$, тогда используя [[лемма (неравенство треугольника для кратчайших путей)|лемму]] получается $v.d = u.d + 1 \ge \delta(s, u) + 1 \ge \delta(s, v)$ 
	- после этого значение $v.d$ больше никогда не изменится
