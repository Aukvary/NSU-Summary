#### связный [[мультиграф]] обладает [[эйлеров цикл|эйлеровым циклом]] тогда и только тогда, когда степень каждой вершины является чётной 

##### доказательство
- **необходимость:**
	- пусть в графе $G$ существует некоторый эйлеров цикл $C$
	- рассмотрим произвольную вершину $v \in V$ 
	- в процессе обхода по циклу каждый раз, когда [[путь]] заходит в вершину $v$ по какому-то ребру, он обязан сращу же покинуть её по другому ребру
	- таким образом, каждое прохождение цикла сквозь вершину $v$ задействует ровно пару инцидентный ей рёбер внося вклад +2 в её степень 
	- если точка является стартовой (конечной), то первое и последние рёбра образуют пару
	- тк каждое ребро задействуется только по разу, то все рёбра у вершин можно разбить по парам $\Rightarrow \deg(v)$ чётная 
- **достаточность:**
	- пусть степени вершин графа чётны
	- начнём с произвольной вершины $a$, проходя по рёбрам в случайном направлении с единственным условием - не использовать отработанные рёбра
	- поскольку степень вершин чётна, мы не может "застрять" в вершине, кроме начальной
	- таким образом образуется простой цикл $C_1$, если он поглотил все рёбра, то задача завершена
	- если остались неиспользованные рёбра цикла $C_1$ из графа
	- в оставшемся графе $H$ степени всех вершин по-прежнему останутся чётными
	- в силу связности графа связности исходного графа $G$, подграф $H$ должен иметь хотя бы одну общую вершину $w$ с циклом $C_1$ 
	- начиная с вершины $w$, повторим процедуру построения цикла для $C_2$ 
	- теперь мы можем объединить циклы начиная с $a$ идя по циклу $C_1$ до вершины $w$, после переключаясь на $C_2$ и потом по $C_1$ до $a$
