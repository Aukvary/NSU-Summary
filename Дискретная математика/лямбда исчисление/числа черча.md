#### числа черча - способ кодирования натуральных чисел с использованием термов

##### кодирование
- 0 - $\lambda fx.x$ 
- 1 - $\lambda fx.f(x)$ 
- 2 - $\lambda fx.f(f(x))$ 

##### операции над числами
- $SUCC$ = $\lambda nfx.f(nfx)$
	- $SUCCn = n + 1$ 
- $PLUS = \lambda mnfx.m\ \ f\ \ (n f x)$ 
	- $PLUS nm = n + m$ 
- $MULT = \lambda mnfx.m(nf)x$ 
	- $MULT nm = n * m$ 
- $EXP = \lambda mnfx.(mn)fx$
	- $EXP n m = n^m$ 
- $PRED = \lambda nf.n(\lambda g.\lambda h.h(g f)) (\lambda u.x) (\lambda .u)$ 
	- $PRED\ 0 = 0$
	- $PRED n = n - 1$ 
- $SUB = \lambda m.(n PRED) m$ 
	- $SUB nm = n - m: n \geq m$ 
	- $SUB nm = 0: n \lt m$
