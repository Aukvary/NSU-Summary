#### устойчивая сортировка - [[сортировка]], которая гарантирует сохранение относительного порядка элементов с одинаковыми ключами

##### пример 
- вы сначала отсортировали список авиарейсов по номеру, а затем устойчивой сортировкой по времени вылета, то рейсы с одинаковым временем останутся упорядоченными по своим номерам
