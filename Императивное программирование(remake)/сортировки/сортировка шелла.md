- улучшенная [[сортировка вставками]]
- сначала мы проходимся по массиву с шагом $h$ постепенно его уменьшая
- тк большая часть массива будет уже отсортирована, когда мы дойдём до $h = 1$, то скорость в худшем случае уменьшится с $N^2$ до $N^{3/2}$ 
- на производительность алгоритма влияет выбор шага(инкремента), например $inc = 2^p - 1$, где $p$ уменьшается с каждым шагом 
```C
void shell_sort(int* nums, int n) {
	for (int h = n / 2; h > 0; h /= 2) {
		for (int i = h; i < n; i++) {
			int temp = nums[i];
			int j;
			for (j = i; j >= h && nums[j - h] > temp; j -= h) {
				nums[j] = nums[j - h];
			}
			nums[j] = temp;
		}
	}
}
```
