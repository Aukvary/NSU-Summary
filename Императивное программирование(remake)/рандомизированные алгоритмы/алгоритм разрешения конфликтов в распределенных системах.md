#### алгоритм
- $n$ процессов конкурируют за доступ к одной базе данных в квантах времени
- в одном кванте успешно может работать только один процесс, иначе все участники попытки блокируются
- каждый процесс независимо пытается обратиться к базе данных в каждом кванте с вероятностью $p = 1/n$
- если попытку совершил ровно один процесс, доступ предоставляется, иначе квант считается потерянным
```C
// концептуальная логика попытки доступа процесса к ресурсу
// n - общее количество конкурирующих процессов
bool try_access_database(int n) {
    // вероятность успеха p = 1/n для минимизации коллизий
    double p = 1.0 / n,
    double random_val = (double)rand() / RAND_MAX,
    return random_val < p,
}
```
