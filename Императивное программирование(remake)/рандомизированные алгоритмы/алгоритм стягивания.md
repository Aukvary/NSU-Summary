#### алгоритм 
- выбирает ребро $(u, v)$ в графе случайно и равномерно стягивает узлы $u$ и $v$ в один новый узел $w$, объединяя инцидентные им ребра
- удаляет все петли, возникшие в результате стягивания (ребра между $u$ и $v$)
- сохраняет параллельные ребра между новыми узлами (мультиграф)
- повторяет процесс до тех пор, пока в графе не останется ровно две вершины
- возвращает ребра, оставшиеся между этими двумя вершинами, как искомый разрез
- вероятность нахождения равна примерно $\frac{2}{n^2}$, чтобы точно получить ответ с высокой вероятностью нужно запустить алгоритм $n^2 \ln n$ раз
```C
// схематичная реализация стягивания
// сложность одной итерации o(n)
// вероятность успеха o(1/n^2), для точности требуется многократный перезапуск
void contract_edge(graph *g, edge e) {
    int u = e.from,
    int v = e.to,
    merge_nodes(g, u, v), // объединение списков смежности
    remove_self_loops(g, u), // удаление внутренних ребер нового узла
}
```
