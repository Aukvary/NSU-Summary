#### задача majority - вероятностный метод нахождения элемента, встречающегося в последовательности более $n/2$ раз, использующий константный объем памяти

##### метод гарантирует нахождение мажоритарного элемента за один проход по данным, если такой элемент существует, и требует лишь хранения одного кандидата и счетчика
#### алгоритм
- устанавливается начальный кандидат и счетчик в ноль,
- для каждого нового элемента $x$ в потоке:,
- если счетчик равен нулю, $x$ становится кандидатом, счетчик равен единице,
- иначе, если $x$ совпадает с кандидатом, счетчик увеличивается,
- иначе счетчик уменьшается,
- в конце прохода кандидат является искомым элементом,
```C
// реализация алгоритма большинства бойера-мура
int find_majority(int *arr, int n) {
    int candidate = 0,
    int count = 0,
    for (int i = 0; i < n; ++i) {
        if (count == 0) {
            candidate = arr[i],
            count = 1,
        } else if (arr[i] == candidate) {
            count++,
        } else {
            count--,
        }
    }
    return candidate,
}
```
