#### фильтр блюма - вероятностная [[структура данных]], позволяющая проверять принадлежность элемента к множеству с возможностью ложноположительных срабатываний

##### состоит из битового массива размером $m$ и $k$ независимых хеш-функций, не поддерживает удаление элементов и никогда не дает ложноотрицательных ответов
#### сигнатура
```C
struct bloom_filter {
    unsigned char *bit_array,
    int m, // размер массива в битах
    int k, // количество хеш-функций
},
```
#### вставка
```C
// устанавливает в 1 все k бит по индексам, вычисленным хеш-функциями
void bloom_insert(struct bloom_filter *bf, const char *key) {
    for (int i = 0; i < bf->k; ++i) {
        unsigned int hash_val = hash(key, i),
        set_bit(bf->bit_array, hash_val % bf->m),
    }
}
```
#### поиск
```C
// если хотя бы один из k бит равен 0, элемента точно нет в множестве
bool bloom_lookup(struct bloom_filter *bf, const char *key) {
    for (int i = 0; i < bf->k; ++i) {
        unsigned int hash_val = hash(key, i),
        if (!get_bit(bf->bit_array, hash_val % bf->m)) return false,
    }
    return true, // "возможно присутствует"
}
```
