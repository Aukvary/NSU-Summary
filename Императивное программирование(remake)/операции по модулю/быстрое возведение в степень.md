- раскладываем значение степени на степени двойки(169 = 128 + 32 + 8 + 1)
- последовательно вычисляем квадраты $a^1, a^2, a^4, ..., a^{64}, a^{128}$
- чтобы получить $a^{169}$ достаточно перемножить $a^{128} \cdot a^{32} \cdot a^{8} \cdot a^{1}$ 
```C
int pow(int num, int p) {
	int res = 1;
	int ptr = 1;
	int a = num;
	
	while (ptr <= p) {
		if (ptr & p) {
			res *= a;
		}
		ptr <<= 1;
		a *= a;
	}
	
	return res;
}
```
