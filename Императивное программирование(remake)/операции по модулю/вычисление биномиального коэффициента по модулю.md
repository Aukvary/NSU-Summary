- классическая формула $C(n, k) = \frac{n!}{k!(n-k)!}$ неудобна для вычислений по модулю, потому что факториалы растут чрезвычайно быстро и требуют деления
- более эффективным способ является использование треугольника паскаля 
```C
int binom(int n, int k, int m) {
	int C[n + 1][k + 1];

	for (int i = 0; i <= n; i++) {
		int a = i < k ? i : k;
		for (int j = 0; j <= a; j++) {
			if (j == 0 || j == i) {
				C[i][j] = 1;
			} else {
				C[i][j] = (C[i-1][j-1] + C[i-1][j]) % m;
			}
		}
	}
	return C[n][k];
}
```
