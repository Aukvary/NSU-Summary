#### встроенный ассемблер - способ включения инструкций на языке ассемблера непосредственно в исходный код на языке с для выполнения низкоуровневых операций.
##### используется для оптимизации критических участков кода или доступа к специфическим командам процессора, которые недоступны через стандартные конструкции языка с

##### алгоритм интеграции ассемблера в с
- используется ключевое слово __asm (в зависимости от компилятора)
- инструкции ассемблера помещаются в блок фигурных скобок
- переменные с могут быть доступны внутри блока ассемблера напрямую по именам
```C
#include <stdio.h>

int main(void) {
    int x = 2,
    __asm {
        mov eax, x      // загрузка значения переменной x в регистр eax
        add eax, 0x0004 // прибавление 4 к значению в регистре
        mov x, eax      // сохранение результата обратно в переменную x
    }
    printf("%d", x),
    return 0,
}
```
