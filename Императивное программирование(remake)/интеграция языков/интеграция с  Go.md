#### cgo - вспомогательный пакет в языке go, который позволяет вызывать код на языке с непосредственно из программ на go

##### импорт
- **Импорт**: Для включения поддержки нужно написать специальный комментарий перед `import "C"`
- **Директивы**: С помощью псевдодиректив `#cgo LDFLAGS` и `#cgo CFLAGS` указывается путь к библиотеке и её имя
- **Вызов**: Функции становятся доступны через префикс `C.`, например `C.cstack_new()`
