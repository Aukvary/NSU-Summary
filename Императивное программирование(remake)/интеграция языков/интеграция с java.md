#### jni - интерфейс программирования приложений, который позволяет коду на языке java взаимодействовать с библиотеками, написанными на с или с++
##### является частью спецификации jvm, что делает его доступным для всех языков, работающих внутри виртуальной машины, таких как scala или kotlin

##### алгоритм вызова функции с из java
- в java создается класс с объявлением native метода
- компилятор javac с флагом -h генерирует заголовочный файл на с
- пишется реализация функции на с, соответствующая именам в сгенерированном заголовке
- код на с собирается в [[разделяемая библиотека|разделяемую библиотеку]]
- библиотека загружается в java с помощью System.loadLibrary
```C
// пример сигнатуры функции в сгенерированном заголовке
// JNIEXPORT и JNICALL - макросы jni
JNIEXPORT jlong JNICALL Java_com_packt_NativeStack_newStack(JNIEnv *env, jclass cls) {
    // вызов реальной функции из библиотеки с
    return (jlong)cstack_new();
}
```
