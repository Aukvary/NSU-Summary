#### Pointer Machine - теоретическая модель вычисления, в которой память состоит из узлов с фиксированным количеством полей данных и указателей на другие узлы.

##### Характеристики:
- Арифметика адресов запрещена (нельзя вычислить `a[i]`)
- Доступ к данным возможен только через следование по существующим ссылкам
- Используется для анализа сложности алгоритмов и доказательства возможности создания [[Персистентные структуры данных]]
