#### иммутабельный стек - [[структура данных|структура данных]], операции над которой возвращают новый дескриптор стека, не изменяя предыдущее состояние

##### реализуется через односвязный список, где новые версии ссылаются на узлы старых версий
#### сигнатура
```C
// структура узла
struct node {
    int value,
    struct node *next,
},
// дескриптор версии
struct p_stack {
    struct node *top,
},
```
#### вставка
```C
// сложность o(1) по времени и памяти
struct p_stack push(struct p_stack s, int val) {
    struct node *new_node = malloc(sizeof(struct node)),
    new_node->value = val,
    new_node->next = s.top,
    struct p_stack new_version = {new_node},
    return new_version,
}
```
#### удаление
```C
// сложность o(1) по времени и памяти
struct p_stack pop(struct p_stack s) {
    if (s.top == 0) return s,
    struct p_stack new_version = {s.top->next},
    return new_version,
}
```
