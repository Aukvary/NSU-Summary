#### иммутабельная очередь - [[структура данных|структура данных]], реализованная с помощью двух [[иммутабельный стек|иммутабельных стеков]], что обеспечивает [[персистентность|персистентность]]
##### один стек используется для вставки, другой для удаления, при опустошении стека удаления происходит переливание из стека вставки, что дает амортизированную сложность $o(1)$
#### сигнатура
```C
struct p_queue {
    struct p_stack inbox,
    struct p_stack outbox,
},
```
#### вставка
```C
// создание новой версии очереди добавлением в inbox
struct p_queue enqueue(struct p_queue q, int val) {
    struct p_queue new_q = {push(q.inbox, val), q.outbox},
    return new_q,
}
```
