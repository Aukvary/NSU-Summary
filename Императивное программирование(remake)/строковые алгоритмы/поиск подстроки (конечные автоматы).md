- для конкретного образца строится детерминированный конечный автомат
- состояния автомата соответствуют длине уже найденного префикса образца
- на каждом шаге, читая символ текста, автомат переходит к новое состояние, которое равно длине самого длинного префикса образца $P$, являющегося суффиксом уже прочитанной части текста 
- если автомат достигает финального состояния, то образец полностью найден
- время поиска $\Theta(n)$, однако требуется фаза предварительной обработки для построения таблицы переходов, которая при простом подходе занимает $O(m^3|\Sigma|)$
