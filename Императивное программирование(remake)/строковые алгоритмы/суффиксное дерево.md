#### суффиксное дерево - сжатое префиксное дерево, содержащее все суффиксы для данной строки

##### свойства
- хранит всевозможные префиксы суффиксов, что эквивалентно хранению всех подстрок
- обычный суффиксный бор, который может занять $n^2$ памяти, суффиксное дерево объединяет цепочки узлов с одним потомком в одно ребро, помечая его подстрокой, благодаря этому дерево занимаем линейное пространство 

##### поиск подстроки
- позволяет найти подстроку $P$ в тексте $T$ за время $O(|P|)$, которое не зависит от длины основного текста 
- поиск образца $P$ начинается из корня дерева, мы просто следуем по рёбрам, символы которых совпадают с символами $P$
- если смогли полностью пройти образец, то подстрока есть в тексте
- все вхождения подстроки можно найти, обойдя поддерево, начинающееся в точке остановки
