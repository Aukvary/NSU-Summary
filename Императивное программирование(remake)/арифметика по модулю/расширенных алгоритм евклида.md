- в отличии от обычного [[алгоритм евклида|алгоритма евклида]], помимо НОД также находит и коэффициенты $x, y$, такие что $ax + by = gcd(a, b)$ 
```C
int gcd(int a, int b, int* x, int* y) {
	if (b == 0) {
		*x = 1;
		*y = 0;
		return a;
	}
	
	int _x, _y;
	
	int d = gcd(b, a % b, &_x, &_y);
	
	*x = _y;
	*y = _x - (a / b) * _y;
	
	return d;
}
```
