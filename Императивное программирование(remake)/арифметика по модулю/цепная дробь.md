#### цепная дробь - математическое выражение вида $$a_0 + \frac{1}{a_1 + \frac{1}{a_2 + ...}}$$ 
##### обозначения
- для краткости используется запись $[a_0; a_1, a_2, ..., a_n, ...]$, где $a_0$ целое число, а остальные коэффициенты $a_i$ (неполные числа) - натуральные числа 

##### типы
- **конечная цепная дробь** - имеет вид $[a_0;a_1, a_2, ..., a_n]$, любое рациональное число можно так представить, притом единственным образом 
- **бесконечная цепная дробь** - содержит бесконечное число элементов, любое иррациональное число можно представить такой дробью

##### нахождение
- находим челую часть числа $a_0$ $[a]$ и остаток $r_0 = \{a\}$ 
- если $r_0 \neq 0$, то вычисляем следующее число $a_1 = 1 / r_0$
- для полученного $a_1$ снова находим целую часть $a_1 = [a_1]$ и остаток $r_1$ 
- если на каком-то шаге остаток равен 0, процесс заканчивается
