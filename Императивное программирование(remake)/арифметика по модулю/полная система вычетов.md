#### полная система вычетов - совокупность чисел, взятых по одному из каждого [[класс эквивалентности|класса эквивалентности]] по модулю $m$ 

##### свойства
- псв всегда содержит ровно $m$ чисел
	- пример для m = 5: $\{0, 1, 2, 3, 4\}$ или $\{-2, -1, 0, 1, 2\}$
