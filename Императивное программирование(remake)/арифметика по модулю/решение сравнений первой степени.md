#### сравнение вида $ax \equiv b\ (\text{mod } m)$ 
##### случай $gcd(a, m) = 1$
- сравнение имеет равно одно решение 
- [[цепная дробь|метод цепных дробей]]
	- дробь $m / a$ разлагается в конечную дробь
	- если $P_{n - 1}$ - числитель её предпоследней подходящей дроби, то решение имеет вид $x \equiv (-1)^{n-1}P_{n-1}b\ (\text{mod } m)$ 
- [[функция эйлера]]
	- $x \equiv b \cdot a^{\phi(m) - 1}\ (\text{mod } m)$ 
##### случай $gcd(a, m) = d > 1$ 
- сравнение имеет решение тогда и только тогда, когда $b$ делится на $d$, иначе решений нет
- делим всё выражение включая модуль на d
	- $\frac{a}{d}x \equiv \frac{b}{d}\ (\text{mod } \frac{m}{d}) = a_1x \equiv b_1\ (\text{mod } m_1)$ 
- тк $gcd(a_1, m_1) = 1$, можно найти единственное решение $x_0$ для нового модуля(случай 1)
- исходное сравнение по модулю $m$ будет иметь ровно $d$ решений вида
	- $x = x_0 + t \cdot m_1, t = 0, 1, ..., d - 1$
