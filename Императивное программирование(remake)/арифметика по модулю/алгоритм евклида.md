- позволяет находить наибольший общий делитель(НОД)
```C
int gcd(int a, int b) {
	return b == 0 ? a : gcd(b, a % b);
}
```

##### теорема ламе
- даёт верхнюю границу количества итераций
- пусть $k$ -число шагов в алгоритме евклида для чисел $n$ и $m$ $n > m$, а $q$ - количество цифр в десятичной записи числа $m$, тогда $k \leq 5q$ 
- **доказательство**
	- если алгоритм состоит из $k$ шагов, то меньшее число $m$ должно быть не меньше $(k + 1)$-го числа фибоначчи 
	- поскольку фибоначчи растут экспоненциально, это ограничивает количество возможных шагов логарифмом от самих числе
