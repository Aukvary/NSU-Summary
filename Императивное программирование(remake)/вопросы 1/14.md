#### Как работают функции getchar() и putchar() для символьного ввода-вывода? Объясните, чем буферизированный ввод отличается от небуферизированного, и приведите пример программы, которая обрабатывает вводимые символы до определённого условия
- [[getchar-putchar]]
- [[буфферизированный ввод]]
```C
#include <stdio.h>

int main(void) {
    char ch;
    while ((ch = getchar()) != '\n')     // пока не встретится конец строки
    {
        if (ch == ' ')                   // если пробел — оставить как есть
            putchar(ch);
        else
            putchar(ch + 1);             // изменить другие символы (сдвиг по ASCII)
    }
    putchar(ch);                         // вывести символ новой строки в конце
    return 0;
}
```
- В этом примере при вводе строки «CALL ME» программа выведет «DBMM NF» только после того, как пользователь нажмет Enter
