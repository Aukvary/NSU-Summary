- предназначен для графов с неотрицательными весами
- поддерживает оценки кратчайшего пути для каждой вершины
- на каждом шаге выбирается вершина с минимальной длинной пути, которая ещё не была посещена 
- для каждой смежной вершины происходит релаксация - смотрим пройти через эту вершину короче, чем тот путь, который мы уже рассматривали, если да, то меняем
- **оценка**
	- зависит от реализации оценки весов
	- массив: $O(V^2)$
	- бинарная пирамида: $O(E\log V)$
	- [[приоритетная очередь|пирамида фибоначчи]]: $O(V\log V + E)$
