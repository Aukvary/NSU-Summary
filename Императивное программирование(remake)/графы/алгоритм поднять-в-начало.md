#### вариация метода проталкивания предпотока со сложностью $o(v^3)$, использующая список вершин и перемещающая вершину в начало списка после каждого [[операция подъёма|подъема]]

##### алгоритм последовательно вызывает процедуру разгрузки (discharge) для вершин списка, пока не будет достигнут конец списка, что означает отсутствие переполненных узлов
```C
// схематичная реализация разгрузки вершины
// сложность всей фазы o(v^3) за счет топологического упорядочивания
void discharge(int u) {
    while (excess[u] > 0) {
        int v = current_neighbor[u],
        if (v == -1) {
            relabel(u),
            current_neighbor[u] = head[u],
        } else if (res_cap(u, v) > 0 && h[u] == h[v] + 1) {
            push(u, v),
        } else {
            current_neighbor[u] = next[v],
        }
    }
}
```
