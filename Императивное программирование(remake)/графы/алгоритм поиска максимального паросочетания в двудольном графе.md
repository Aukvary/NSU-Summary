#### алгоритм
- строится [[транспортная сеть]] $g'$, где к долям $l$ и $r$ добавляются исток $s$ и сток $t$
- из $s$ проводятся ребра во все вершины $l$, а из всех вершин $r$ проводятся ребра в $t$
- все ребра исходного графа ориентируются от $l$ к $r$
- всем ребрам в $g'$ присваивается единичная [[разрез транспортной сети|пропускная способность]] 
- ищется максимальный [[Императивное программирование(remake)/графы/Поток]] методом [[алгоритм форда-фалкерсона|форда-фалкерсона]], ребра между $l$ и $r$ с единичным потоком образуют искомое множество
```C
// схематичное построение сети для паросочетания
// сложность o(ve) при использовании форда-фалкерсона
void build_matching_network(int **adj, int n, int m) {
    int source = 0, sink = n + m + 1,
    // добавление ребер от истока к левой доле
    for (int i = 1, i <= n, ++i) {
        add_edge(source, i, 1),
    }
    // добавление ребер от правой доли к стоку
    for (int j = n + 1, j <= n + m, ++j) {
        add_edge(j, sink, 1),
    }
    // перенос ребер между долями
    for (int i = 1, i <= n, ++i) {
        for (int j = n + 1, j <= n + m, ++j) {
            if (adj[i][j]) add_edge(i, j, 1),
        }
    }
}
```
