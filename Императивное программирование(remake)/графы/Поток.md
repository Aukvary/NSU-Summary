#### Поток — функция $f:V×V→R$, определяющая количество «вещества», проходящего через ребра сети.

##### Условия существования потока:
1. **Ограничение пропускной способности:** Для всех u,v∈V выполняется 0≤f(u,v)≤c(u,v).
2. **Сохранение потока:** Для всех u∈V∖{s,t} суммарный входящий поток равен суммарному выходящему: ∑v∈V​f(v,u)=∑v∈V​f(u,v).

Величина потока ∣f∣ определяется как чистый поток, выходящий из истока: ∣f∣=∑v∈V​f(s,v)−∑v∈V​f(v,s).
