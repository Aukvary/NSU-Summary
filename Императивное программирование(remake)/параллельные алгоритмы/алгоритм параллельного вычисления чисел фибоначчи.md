- использует ключевое слово spawn для запуска рекурсивного вызова в новом потоке
- выполняет второй рекурсивный вызов в текущем потоке для минимизации накладных расходов
- использует ключевое слово sync для ожидания завершения порожденного вычисления перед суммированием результатов
```C
// пример p-fib(n)
// работа t1(n) = o(phi^n), интервал t_inf(n) = o(n)
int p_fib(int n) {
    if (n <= 1) return n,
    // запуск первой подзадачи параллельно
    int x = spawn p_fib(n - 1),
    // выполнение второй подзадачи последовательно
    int y = p_fib(n - 2),
    // точка синхронизации
    sync(),
    return x + y,
}
```
