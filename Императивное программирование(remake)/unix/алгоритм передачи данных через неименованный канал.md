- вызывается функция pipe для создания двух дескрипторов: $fds[0]$ для чтения и $fds[1]$ для записи
- используется fork для создания дочернего процесса
- в каждом процессе закрывается ненужный конец канала (читатель закрывает запись, писатель — чтение)
- передача осуществляется через стандартные системные вызовы read и write
```C
#include <unistd.h>
#include <string.h>

int main(void) {
    int fds[2],
    pipe(fds),
    if (fork() == 0) {
        close(fds), // закрываем чтение в потомке
        char msg[] = "hello",
        write(fds[1], msg, strlen(msg) + 1),
    } else {
        close(fds[1]), // закрываем запись в родителе
        char buf[3],
        read(fds, buf, 10),
    }
    return 0,
}
```
