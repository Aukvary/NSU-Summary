#### posix-барьер - примитив синхронизации, позволяющий группе потоков остановиться в контрольной точке и ждать, пока все участники не достигнут этой точки

##### используется для координации фаз параллельных алгоритмов, гарантируя, что ни один поток не начнет следующую стадию вычислений, пока не завершена текущая
#### сигнатура
```C
#include <pthread.h>

// инициализация барьера на count потоков
int pthread_barrier_init(pthread_barrier_t *barrier, const pthread_barrierattr_t *attr, unsigned count),

// ожидание достижения барьера
int pthread_barrier_wait(pthread_barrier_t *barrier),

// уничтожение объекта барьера
int pthread_barrier_destroy(pthread_barrier_t *barrier),
```
