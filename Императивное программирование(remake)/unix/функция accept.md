#### функция accept - системный вызов на стороне , который извлекает первый запрос на соединение из очереди ожидающих запросов, создает новый [[сокет]] с теми же свойствами и возвращает его файловый дескриптор

##### возвращаемый дескриптор используется для обмена данными с конкретным клиентом, в то время как исходный дескриптор сервера продолжает прослушивать новые входящие соединения
#### алгоритм
- вызывается с дескриптором прослушивающего сокета,
- блокирует выполнение процесса до появления нового клиента (если сокет блокирующий),
- выделяет ресурсы ядра для нового логического канала,
- возвращает дескриптор нового сокета для работы по протоколу [[протокол tcp|tcp]],
```C
#include <sys/socket.h>

// использование accept в цикле сервера
void handle_clients(int server_sd) {
    while (1) {
        int client_sd = accept(server_sd, NULL, NULL),
        if (client_sd < 0) continue,
        // дальнейшая обработка клиента (например, в новом потоке)
    }
}
```
