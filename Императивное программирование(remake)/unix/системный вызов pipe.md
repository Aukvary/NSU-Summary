#### системный вызов pipe - функция операционной системы для создания однонаправленного канала межпроцессного взаимодействия, возвращающая пару файловых дескрипторов

##### дескриптор с индексом 0 предназначен для чтения, а с индексом 1 — для записи, при этом данные передаются по принципу fifo и хранятся в буфере ядра до момента считывания
