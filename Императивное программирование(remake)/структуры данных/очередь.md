#### очередь - линейная структура данных, организованная по принципу FIFO (First-In-First-Out — «первым пришёл, первым вышел»)
#### сигнатура
```C
int queue[SIZE];
int head = 0;
int tail = 0;
```
#### вставка
```C
void push(int val) {
	queue[tail++] = val;
}
```
#### поиск
```C
int peek() {
	return queue[head];
}
```
#### удаление
```C
int pop() {
	return queue[head++];
} 
```
