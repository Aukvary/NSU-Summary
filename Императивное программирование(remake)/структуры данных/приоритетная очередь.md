#### приоритетная очередь - структура данных, в которой каждому элементу назначается определённый приоритет, при извлечении сначала идёт элемент с большим приоритетом, если приоритеты совпадают, то сначала идёт тот, который был добавлен раньше

##### реализации
- **отсортированный список**
	- при добавлении ищется подходящее место, например через бинарный поиск и на нужное место вставляется элемент
	- при удалении просто берётся первый элемент 
- **несортированный список**
	- элемент добавляется в конец
	- при удалении просматривается весь список и ищется элемент с наибольшим приоритетом
- **бинарные кучи**
```C
int* heap;
int n;

void heapify(int i) {
	int l = i * 2 + 1;
	int r = i * 2 + 2;
	  
	int idx = i;
	
	if (l < n && heap[l] < heap[idx]) {
		idx = l;
	}  
	
	if (r < n && heap[r] < heap[idx]) {
		idx = r;
	}
	
	if (i != idx) {
		swap(&heap[i], &heap[idx]);
		headpify(idx);
	}
}  

void push(int num) {
	heap[n] = num;
	
	for (int i = n; i > 0;) {
		int parent = (i - 1) / 2;
		
		if (heap[i] < heap[parent]) {
			swap(&heap[i], &heap[parent]);
			i = parent;
		} else break;
	}
	n++;

}

void pop() {
	swap(&heap[0], &heap[n - 1]);
	heapify(0);
	return heap[--n];
}
```
