#### кольцевая очередь - [[очередь]], в которой указатели перемещаются циклически 
#### сигнатура
```C
int queue[SIZE];
int head = 0;
int tail = 0;
```
#### вставка
```C
void push(int val) {
	queue[tail++ % SIZE] = val;
}
```
#### поиск
```C
int peek() {
	return queue[head];
}
```
#### удаление
```C
int pop() {
	return queue[head++ % SIZE];
}
```
