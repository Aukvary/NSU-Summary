#### Стек — это линейная структура данных, в которой элементы хранятся в упорядоченном виде. Основной принцип организации стека называется LIFO (Last-In-First-Out — «последним пришёл, первым вышел»): это означает, что элемент, вставленный последним, будет извлечён самым первым
##### сигнатура
```C
int stack[SIZE];
int ptr = -1;
```
##### вставка
```C
void push(int val) {
	stack[++ptr] = val;
}
```
##### поиск
```C
int peek() {
	if (ptr == -1) { 
		exit(1); 
	}
	
	return stack[ptr];
}
```
##### удаление
```C
int pop() {
	if (ptr == -1) {
		exit(1);
	}
	
	return stack[ptr--];
}
```
