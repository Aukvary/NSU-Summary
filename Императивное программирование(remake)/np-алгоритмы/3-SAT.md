#### 3-SAT - частный случай [[задача SAT|задачи SAT]], в котором каждое условие содержит ровно 3 литерала, соответствующих разным переменным

##### свойства
- несмотря не ограничения сложность совпадает с SAT и задача является [[np-полная задача|NP-полной]] 

##### сведение к задаче о независимом множестве
- для каждого условия строится треугольник с тремя вершинами
- вершины треугольника помечаются литералами из соответствующего условия
- между вершинами разных треугольников проводятся рёбра конфликта, если их метки противоречат друг другу (в одном $x$ в другом $\bar x$)
- целевой размер независимого множества устанавливается равным $K = k$(количеству условий)
- поскольку в треугольнике все вершины соединены, из каждого треугольника можно выбрать не более одного узла
- если формула выполнима, мы можем выбрать в каждом треугольнике по одному истинному литералу, тк мы не выбираем конфликтующие литералы, эти узлы не будут соединены рёбрами конфликта, образуя независимое множество размера $k$
- обратно: независимое множество размера $k$ обязано содержать ровно по одному узлу из каждого треугольника, и отсутствие рёбер между ними гарантирует отсутствие логических противоречий присваивания
