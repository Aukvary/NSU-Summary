#### сумма подмножества - поиск подмножества чисел, сумма которых равна $W$

##### свойства
- она является [[np-полная задача|NP-полной]], тк время работы динамического программирования $O(nW)$ зависит от величины числа, которая растёт экспоненциально относительно размера входа в битах

##### сведение от вершинного покрытия (Base-4)
- для доказательства используется перевод экземпляра задачи о вершинном покрытии (граф $G$, число $k$) в набор чисел $X$ и целую сумму $T$
- **построение чисел**
	- каждое число представляется как $(E + 1)$-разрядное в системе счисления по основанию 4, где $E$ - число ребро 
	- **числа рёбер:** для каждого ребра $i$ создаётся число $b_i = 4^i$ 
	- **числа вершин:** для каждой вершины $v$ создаётся число $a_v = 4^E + \Sigma_{i \in \Delta(v)}$, где $\Delta(v)$ - рёбра
	- целевая сумма $T: T = k \cdot 4^E + \Sigma_{i = 0}^{E-1} 2 \cdot 4^i$ 
- **почему это работает**
	- **контрольный размер покрытия:** самый старший разряд (коэффициент при $4^E$) в целевой сумме равен $k$, тк этот разряд есть только в числах вершин, это ограничивает количество вершин числом $k$
	- **покрытие рёбер:** каждый из $E$ младших разрядов в целевой сумме $T$ равен 2
