#### вершинное покрытие - подмножество вершин графа, такое, что каждое ребро графа инцидентно хотя бы одной вершине из этого подмножества

##### концептуально задача вершинного покрытия является задачей минимизации ресурсов для «накрытия» всех связей в графе

##### задача о локальном минимуме
- **пространство решений:** все подмножества вершин, являющиеся вершинными покрытиями
- **функция стоимости:** размер подмножества, мы хотим минимизировать это количество
- **градиентный спуск:** начинаем с полного множества вершин и на каждом шаге удаляем вершину, если результат остаётся верным покрытием, если нельзя удалить вершину, то мы в локальном минимуме 
- **проблема пути:**
	- пусть у нас есть граф-путь с нечётным количеством узлов
	- **глобальный оптимум:** состоит из всех узлов с чётными индексами
	- **плохой локальный минимум:** у нас есть покрытие, где пропущен каждый 3 узел, это покрытие значительно больше оптимума, но из нельзя удалить ни один узел, сохранив покрытие рёбер
