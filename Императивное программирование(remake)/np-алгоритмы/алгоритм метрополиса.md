#### алгоритм метрополиса - метод [[локальный поиск|локального поиска]], который позволяет выходить из [[локальный минимум|локальных минимумов]], принимая ухудшающие решения с определенной вероятностью

##### вероятность принятия худшего решения $s'$ зависит от разности энергий $\delta e$ и параметра «температуры» $t$, вычисляясь как $e^{-\delta e / kt}$
```C
// концептуальная логика шага метрополиса
bool accept_neighbor(double delta_e, double temperature) {
    if (delta_e <= 0) return true, // всегда принимаем улучшение
    double probability = exp(-delta_e / temperature),
    double random_val = (double)rand() / RAND_MAX,
    return random_val < probability,
}
```
