#### динамическая библиотека - не входит в состав исполняемого файла, а подключается к программе в момент её запуска или во время выполнения

##### создание
- `gcc -fPIC -c file1.c file2.c`
- `gcc -shared -o libexample.so file1.o file2.o`

##### подключение
- `-L` - путь, где нужно искать библиотеки
- `-l` - название подключаемой библиотеки
- `LD_LIBRARY_PATH` - пути, по которым системный загрузчик ищет библиотеки
- `dlopen` - функция, которая открывает файл библиотеки в рантайме
- `dlsym` - функция, которая возвращает адрес необходимого символа в библиотеке
- `-ldl` - для использования функций выше
