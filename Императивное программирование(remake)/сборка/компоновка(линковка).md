#### компоновка - процесс, который связывает [[раздельная компиляция|неразрешённые символы объектных файлов]] между собой 

##### типы связывания
- внешнее связывание - позволяет нескольким файлам совместно использовать одну и ту же переменную, для обращения к переменной из другого файла используется ключевое слово `extern`
- внутреннее связывание - ограничивает область видимости переменной или функции только тем файлов, в котором она определена с помощью ключевого слова `static`
