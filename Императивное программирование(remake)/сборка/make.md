#### make - инструмент автоматизации сборки программ, который управляет зависимостями и минимизирует объем перекомпиляции кода путем отслеживания меток времени изменения файлов

##### используется для эффективной сборки больших проектов на языке с, где изменение одного заголовочного файла требует пересборки только зависимых от него объектных файлов
##### алгоритм работы утилиты make
- считывает инструкции из файла makefile
- строит дерево зависимостей между исходными файлами, объектными файлами и исполняемым файлом
- сравнивает время последнего изменения цели и её зависимостей
- выполняет команды компиляции только для тех целей, чьи зависимости новее самой цели
```make
target: dependencies
[tab] command

out/main.o: main.c
     gcc -c main.c -o out/main.o
```
