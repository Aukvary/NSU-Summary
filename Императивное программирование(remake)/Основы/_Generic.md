#### _ Generic - макрос, который позволяет создавать обобщённые макросы, то есть макросы, которые будут подстраиваться под тип аргумента

##### пример
```C
#define GENERIC_MACRO(x) _Generic((x),  \
	int: int_func						\
	float: float_func                   \
	default: default_func               \
)(x)
```
