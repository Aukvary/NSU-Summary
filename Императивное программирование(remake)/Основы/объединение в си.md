#### объединение - специальный тип данных, который позволяет хранить в одном и том же месте памяти значения разных типов 

##### свойства
- поля объединения накладываются друг на друга
- размер объединения равняется размеру самого большого элемента 

```C
union MyUnion {
	char* field0;
	int field1;
	char field2;
}

MyUnion v1 = { "aga aga" };
MyUnion v2 = { 1000 };
MyUnion v3 = { 'a' };
```
