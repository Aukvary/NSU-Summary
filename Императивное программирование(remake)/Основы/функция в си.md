#### функция в [[язык си|си]] - это самодостаточная единица программного кода, спроектированная для выполнения определённой отдельной задачи

##### Основные причины использования функций:
- **Исключение дублирования:** они избавляют от необходимости многократно писать один и тот же код для повторяющихся действий.
- **Повторное использование:** однажды написанную функцию можно применять в разных частях одной программы или даже в разных проектах.
- **Модульность:** функции делают программу более структурированной, что существенно улучшает её читабельность, упрощает поиск ошибок и внесение изменений.

##### типы 
- **Функции с возвращаемым значением:** они вычисляют результат и передают его обратно в вызывающую программу с помощью оператора `return`. Тип такой функции должен совпадать с типом возвращаемого ею значения (например, `int`, `double`)
- **Функции без возвращаемого значения(void):** используются для выполнения действий (например, вывода текста), не требующих возврата конкретного числа или символа
- **Функции без параметров:** если функция не принимает данных, в скобках указывается ключевое слово `void` (например, `int main(void)`)
- **Рекурсивные функции:** функции, которые вызывают сами себя

##### передача аргумента
- **по значению** - функцию передаётся **копия** значения переменной, поэтому любые изменения этой копии внутри функции не влияют на оригинал в вызывающей программе
	- главный плюс - безопасность: функция работает с копией
	- при передачи структуры она полностью копируется
- **по указателю** - Если программе необходимо, чтобы функция изменила значение переменной в вызывающей функции, передаётся её адрес (указатель). В этом случае функция работает напрямую с областью памяти оригинальной переменной через операцию разыменования. 
	- [[массив в си|массивы]] всегда передаются по ссылке
	- высокая эффективность при передаче больших значений, тк мы их не копируем, а передаём только указатель
