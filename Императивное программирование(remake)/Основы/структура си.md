#### структура - составной тип данных, который позволяет объединять под одним именем переменные разных типов

##### объявление
```C
struct MyStruct { // опеределяем шаблон структуры
	int field0;
	char field1;
	
	struct { // вложенная структура анонимная структура 
		char* str;
	} inside_struct;
};

struct MyStruct var; // переменная с типом структуры

// доступ к полям 
var.field0 = 20;
char a = var.field1;
```

##### инициализация структуры
- классический подход
	- значения должны идти в том же порядке, что и члены структуры
	- если инициализируется структура со статической продолжительностью хранения, значения должны быть константными выражениями  
```C
struct MyStruct var = { 10, 20 };
```
- назначенные инициализаторы 
	- используется точка перед именем поля
```C
struct MyStruct var = { .field1 = 10 };
```

##### указатель на структуру
- для доступа к полям указателя на структуру используется оператор `->`, который эквивалентен разыменованию и обращению к полю
```C
struct MyStruct var = {
	.inside_struct = {
		.str = "str",
	},
};

struct MyStruct* ptr = &var;
ptr->field0 = 20; // тоже самое что и (*ptr).field0 = 20;
```

##### битовые поля
- позволяет устанавливать размер поля в структуре 
```C
struct MyBitSruct {
	int field0:    31; //будет занимать 31 бит
	uin8_t field1: 1;  //будет занимать 1 бит
};
```

##### составные литералы 
```C
void func(struct MyStruct s);

func((struct MyStruct){
	.field0 = 0,
	.field1 = 'A',
});
```

##### гибкий массив
- позволяет создавать структуру, размер которой может варьироваться в зависимости от объёма данных в последнем поле 
- в структуре обязательно быть любое другое поле 
- массив необходимо объявлять с пустыми скобками
```C
struct FlexArray {
	int count;
	int arr[];
}

struct FlexArray vec1 = malloc(sizeof(FlexArray) + sizeof(int) * 10);
struct FlexArray vec2 = malloc(sizeof(FlexArray) + sizeof(int) * 20);
```
