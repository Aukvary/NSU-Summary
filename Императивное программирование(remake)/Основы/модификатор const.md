#### модификатор const - модификатор, который запрещает программисту менять значение переменной 

##### особенности
- `const float* ptr` - указатель на константу, то есть мы не можешь менять данных, но можешь менять адрес
- `float* const ptr` - константный указатель, то есть мы можешь менять данные, но не можешь менять адрес
- `const float* const ptr` - в данном случае нельзя менять ни адрес, ни данные
