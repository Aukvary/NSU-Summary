#### массив - линейный набор элементов одного типа данных, которые хранятся в последовательных ячейках памяти

##### объявление
- `type name[count] = {//...};` - `int nums[] = {1, 2, 3, 4}`
- если начальных значений меньше чем количество, то они заполнятся нулями `int nums[5] = {1, 2}`
- можно инициализировать конкретные значения `int nums[5] = {[2] = 10, [4] = 2}`
##### многомерные массивы
- объявление `float nums[2][3] = {{0.1f, 0.2f, 0.3f}, {0.4f, 0.5f, 0.6f}}` - массив из 2 элементов, где каждый элемент это массив из 3 элементов, для обращения к элементу `0.4f` используется операция `nums[1][0]`

##### свойства
- **Доступ через указатели:** Выражение `days[index]` полностью эквивалентно разыменованию указателя со смещением: `*(days + index)` 
- **Арифметика указателей:** При добавлении единицы к указателю на массив, адрес увеличивается не на 1 байт, а на размер типа данных, на который он указывает. Например, для типа `short` адрес увеличится на 2 байта, а для `double` — на 8 байт
