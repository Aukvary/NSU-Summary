#### стековый фрейм - структура, которая кладётся в стек при вызове функции

##### содержимое 
- аргументы функции
- локальные переменные 
- адрес возврата

##### уязвимости
- возвращение адреса локальной переменной 
	- локальные переменные существуют только пока активна их область видимости
- переполнение буфера
	- если программа записывает в массив на стеке больше данных, чем было выделено, лишние байты начинают перезаписывать соседние данные в стековом сегменте
	- можно специально подобрать данные так, чтобы перезаписать адрес возврата и перехватить управление процессом
