#### функция с переменных количеством аргументов - [[функция в си|функция]], которая способна принять любое количество аргументов

##### компоненты
- `stdarg` - [[заголовочный файл]], который содержит необходимый функционал 
- `...` - указывается в параметрах функции, чтобы использовать неопределённое количество аргументов
- `va_list` - специальный тип данных, предназначенный для хранения информации о списке аргументов
- `va_start(ap, parmN)` - инициализирует переменную типа `va_list`, первым аргументом передаётся список, а вторым имя последнего аргумента 
- `va_arg(ap, type)` - последовательно извлекает аргументы, при каждом вызове он возвращает значение текущего аргумента указанного типа и перемещает указатель к следующему элементу в списке
- `va_copy(dst, src)` - копирует текущее состояние списка, если придётся пройтись несколько раз
- `va_end(ap)` - выполняет необходимую очистку после обработки аргументов
