##### функции для работы с файлами
- `FILE* fopen(const char *filename, const char *mode)` - открывает файл и настраивает буфер для работы с ним
	- `"r"` - чтение
	- `"w"` - перезапись
	- `"a"` - дополнение 
	- `"...b"` - битовое чтение/битовая запись
- `fclose(FILE *fp)` - закрывает файл и сбрасывает все связанные с ним буферы
	- если всё прошло успешно возвращает `0`, иначе `EOF`
- `fread(), fwrite()` - двоичный ввод-вывод, они работают с блоками памяти фиксированного размера и копируют данные как они есть
- `fseek(FILE *fp, long offset, int origin)` - устанавливает указатель на текущей позиции в файле на заданное смещение `offset` относительно точки `origin` 
- `ftell(FILE *fp)` - возвращает текущее значение позиции указателя файла в байтах от его начала
