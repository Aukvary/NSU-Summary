#### цикл while - цикл с предусловием. Это значит, что проверка условия выполняется до начала выполнения тела цикла

##### свойства
- `break` - прерывает цикл
- `continue` - сразу переходит на следующую итерацию 

##### пример использования
```C
while (scanf("%d", &var) != 0) { //считываем числа, пока пока можем 
	//...
}
```
