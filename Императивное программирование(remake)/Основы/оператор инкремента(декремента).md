#### оператор инкремента(декремента) - оператор увлечения(уменьшения) значения на 1

##### обозначения
- ++(--)

##### свойства
- префиксное применение изменяет значение до его использования, а постфиксное после
```C
int a = 5;
int b = a++; //b = 5, a = 6
itn c = ++b; //c = 6, b = 7
```
