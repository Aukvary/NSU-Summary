#### scanf-printf - функции для форматированного ввода/вывода 

##### свойства
- находятся в [[заголовочный файл|заголовочном файле]] `stdio.h` 
##### спецификаторы
- для целых чисел
	- `%d/%i` - Знаковое десятичное целое число
	- `%u` - Беззнаковое десятичное целое число
	- `%o`
	- `%x/%X` - Шестнадцатеричное число (в нижнем/верхнем регистре)
- для чисел с плавающей запятой
	- `%f` - число с плавающей точкой(float)
	- `%e/%E` - экспоненциальная нотация(1e-5)
	- `%g/%G` - автоматический выбор между `%f` и `%e` в зависимости от значения
- для символов и строк
	- `%c` - одиночный символ
	- `%s` - строка
- прочее
	- `%p` — адрес (указатель)
	- `%%` — вывод самого знака процента

##### модификаторы ширины и точности
- общий шаблон `%[флаг][ширина][.точность]спецификатор`
- **ширина** - минимальное количество символов, которое займёт вывод. Если число короче, оно дополняется пробелами
- **точность** 
	- для `%f`, `%e` - количество знаков после запятой.
	- для `%g` - максимальное количество значащих цифр.
	- для `%s` - максимальное количество выводимых символов строки.
	- для целых чисел - минимальное количество цифр (с ведущими нулями).
- **флаги**
	- `-` - выравнивание по левому краю
	- `+` - всегда выводить знак (+/-) перед числом
	- ` ` - выводить пробел перед положительным числом (резервирует место под знак)
	- `#` - альтернативная форма (например, добавляет `0x` для шестнадцатеричных чисел)
	- `0` - заполнение пустых мест в поле ведущими нулями вместо пробелов
	- `*`
		- printf - можно передавать ширину и точность как аргументы `printf("Bec = %*.*f\n", 8, 3, weight);`
		- scanf - заставляет функцию прочитать значение, но не присваивать его переменной `scanf(%*d)`
- **модификатор типа**
	- `h` — для `short int`.
	- `l` — для `long int`.
	- `ll` — для `long long int`.
	- `L` — для `long double`.
	- `z` — для типа `size_t` (результат `sizeof`).
