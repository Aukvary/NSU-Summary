#### директива include - инструкция препроцессора, которая сообщает компилятору о необходимости включить содержимое другого файла в текущий исходный текст

##### свойства
- позволяет совместно использовать информацию между файлами
- пример [[директива препроцессора|директивы препроцессора]] 
- обычно используется для подключения заголовочных файлов с расширением .h
