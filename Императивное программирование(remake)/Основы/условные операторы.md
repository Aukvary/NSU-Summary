#### условные операторы - операторы, которые служат для пророк условий

##### if-else
- последовательно проверяет все условия
- если условие не равно 0, то условие считается верным
```C
if (cond) { // основное условие

} else if (cond) { // вспомогательнрое условие

} else { // если не одной условие не прошло
	
}
```
##### switch
- работает быстрее, чем if-else
- если не писать `break`, то код продолжит выполняется со следующей метки
```C
switch (var) {
	case 0: {
		//..
		break;
	}
	
	case 1: {
		//..
		break;
	}	
	
	//...
	
	default: { //выполняется, если остальные состояния не подошли
		break;
	}
}
```
