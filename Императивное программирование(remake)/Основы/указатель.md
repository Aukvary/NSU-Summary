#### указатель - переменная, значение которой хранит адрес в памяти другой переменной 

##### объявление 
- `*` - возле типа обозначает, что это указатель на этот тип, а также используется для разыменования, то есть получения самой переменной
- `&` - возле переменной обозначает её адрес
```C
int a = 10;
int* ptr = &a;
*ptr = 20;
```
