#### мультикуча - [[структура данных|структура данных]] для обработки графов вычислений во [[внешняя память|внешней памяти]], в которой ключом является номер вершины, а значением — данные, передаваемые по ребру, позволяющая эффективно извлекать все входящие сообщения для конкретной вершины

##### обеспечивает амортизированную сложность операций $o((1/b) \log_{m/b} (n/b))$, что позволяет выполнять расчеты над огромными ориентированными ациклическими графами (dag) за время, сравнимое с сортировкой данных
#### сигнатура
```C
// концептуальная структура элемента кучи
struct heap_item {
    uint64_t vertex_id, // номер вершины (ключ)
    void *data,         // передаваемое по ребру значение
},
```
