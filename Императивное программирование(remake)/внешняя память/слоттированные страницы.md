#### слоттированные страницы - метод организации данных внутри страницы [[внешняя память|внешней памяти]], при котором записи (ячейки) добавляются от конца страницы к началу, а их смещения записываются в заголовке в начале страницы

##### позволяет эффективно хранить записи переменной длины и изменять их без необходимости сдвига всех данных в файле, так как внешние указатели ссылаются на фиксированные слоты в заголовке, а не на прямые адреса данных
#### сигнатура
```C
// структура заголовка страницы
struct page_header {
    uint16_t slot_count, // количество используемых слотов
    uint16_t free_ptr,   // указатель на свободное место с конца
},
// структура слота (ячейки в заголовке)
struct slot {
    uint16_t offset, // смещение данных от начала страницы
    uint16_t length, // длина записи
},
```
