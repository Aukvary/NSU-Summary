- дан массив чисел $A = [a_0, a_1, ..., a_{n - 1}]$, требуется находить значение/индекс минимального элемента на произвольном промежутке 

##### сведение RMQ к LCA
- на основе массива строится [[декартово дерево]]
- значения используются как приоритеты, а индексы как ключи
- результат RMQ на отрезке $[i, j]$ будет в точности соответствовать LCA узлов с индексами $i$ и $j$ в построенном декартовом дереве

##### метод разреженных таблиц 
- препроцессинг
	- вычисляются минимумы для всех возможных отрезков, длина которых является степенью двойки ($s^k$) 
	- минимум на отрезке длины $2^k$ находится как минимум из двух перекрывающихся отрезков длинны $2^{k-1}$ 
	- время $O(n\log n)$, память $O(n\log n)$ 
	- если все соседние элементы отличаются ровно на 1(такой массив получается при сведении LCA к RMQ), то можно ускорить алгоритм
		- массив делится на блоки размера $b = \frac{1}{2}\log n$ 
		- для минимумов блоков строится разреженная таблица
		- внутри каждого блока минимум ищется с помощью заранее вычисленных таблиц 
		- скорость возрастает до $O(n)$ 
- запрос 
	- любой отрезок произвольной длины можно покрыть двумя блоками, длины который равны наибольшей степени двойки, не превосходящей длину отрезка
	- ответ находится за $O(1)$ путём сравнения этих значений
