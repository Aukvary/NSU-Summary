#### выпуклая оболочка - наименьший выпуклый многоугольник, который содержит все точки множества $Q$ 

##### сканирование по грэхему
- **выбор базы:** находится точка $p_0$ с минимальной координатой $y$ 
- **сортировка:** все остальные точки сортируют по возрастанию полярного угла относительно $p_0$ 
- **обход:** точки поочерёдно добавляются в [[Императивное программирование(remake)/структуры данных/стек|стек]] 
	- если точка образует с двумя верхними точками стека "правый поворот"(или идёт прямо), верхняя точка удаляется из стека, пока не будет левого поворот 
- $O(n\log n)$ 
##### алгоритм джарвиса
- **выбор базы:** начинается с самой нижней точки $p_0$, которая гарантированно входит в оболочку
- **поиск следующей вершины:** из текущей точки выбирается та, у которой минимальным полярный угол по отношению ко всем точкам 
- **завершение:** процесс повторяется, пока алгоритм не вернётся в исходную точку $p_0$ 
- $O(nh)$, где $h$ количество точек в оболочке
