- **Числа Ферма**: Пьер Ферма предположил, что числа вида $Fn​=2^{2^n}+1$ всегда простые. Это верно для n=0,1,2,3,4, но уже Эйлер показал, что $F_5​$ — составное число. На данный момент не найдено ни одного простого числа Ферма для n>4.
- **Многочлен Эйлера**: Формула $f(n)=n^2+n+41$ выдаёт простые числа для всех n от 0 до 39. Существуют модификации этой формулы (например, $n^2−79n+1601$), которые дают простые числа для n от 0 до 79, но они часто повторяются.
- **Теорема о невозможности**: Доказано, что не существует непостоянного многочлена с целыми коэффициентами, который давал бы только простые числа для всех натуральных n.
- **Формулы Уилланса**: Существуют математически корректные формулы для вычисления n-го простого числа, использующие тригонометрические функции и факториалы, но они **крайне неэффективны** на практике из-за колоссального числа операций.
