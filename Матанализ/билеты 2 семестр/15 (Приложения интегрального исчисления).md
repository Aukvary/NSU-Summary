#### Приложения интегрального исчисления
##### задача 1
###### условие 
Электромотор имеет $P = 100$Вт, при работе он нагревается и $P$ падает по закону $P(t) = 40 + \frac{60}{t + 1}$ в минутах, какое количество энергии будет потрачено за 4 минуты
###### решение
$E = PT$
$\Delta E  = P(t)dt$ 
$E = \int_0^4(40 + \frac{40}{t+1})dt = 40t|_0^4 + 60 \ln(t+1)|_0^4$ 
$E = 160 + 60\ln 5 \approx 256$ Вт$\cdot$мин $\approx 156$ кДЖ
##### задача 2
###### условие 
стержень длиной $L$ имеет $\rho(x) = \rho_0(1 + \frac{x^2}{L^2})$, найдите положение центра тяжести 
###### решение 
условие равновесия $l_2\Delta m_2 = l_1\Delta m_1$ 
$(x_c - x_2)\rho(x)\Delta m_2 = (x_1 - x_c) \rho(x)\Delta m_1$ 
$(x_c - x_2)\rho(x)\Delta x = (x_1 - x_c) \rho(x)\Delta x$ 
$\int_0^{x_c}(x_c - x)\rho(x)d x = \int_{x_c}^L(x - x_c)\rho(x)dx$ 
$\int_0^{x_c} \rho(x)dx - \int_0^{x_c}x\rho(x)dx = \int_{x_c}^L x\rho(x)dx - \int_{x_c}^L x_c \rho(x)dx$
$x_c(\int_0^{x_c} \rho(x)dx + \int_{x_c}^L \rho(x)dx) = \int_{x_c}^L x\rho(x)dx + \int_o^{x_c} x\rho(x)dx$  
$x_c = \frac{\int_0^L x\rho(x)dx}{\int_0^L \rho(x)dx}$ 
$M = \int_0^L\rho(x)dx = \rho_0 \int_0^L (1 + \frac{x^2}{L^2})dx = \rho(L + \frac{L}{3}) = \frac{4}{3}\rho_0L$  
$\int_0^L x\rho(x)dx \rho_0 \int_0^L xdx + \rho_0 \int_0^L \frac{x^3}{L^2}dx = \rho_0 \frac{L^2}{4} + \rho_0 \frac{L^2}{4} = \rho \frac{3}{4}L^2$ 
$x_c =\frac{\frac{3}{4}L^2}{\frac{3}{4}L} = \frac{9}{16}L$
