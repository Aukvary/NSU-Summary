#### комплексные числа - числа вида $z = a + bi$
- $a = ReZ$ - действительная часть
- $b = InZ$ - мнимая часть
- $z = a + bi$ - алгебраическая форма записи
- $z = \varrho(\cos x + i\sin x)$ - тригонометрическая запись 
	- где $\varrho$ длинна вектора
	- $x$ - угол, который образует вектор
