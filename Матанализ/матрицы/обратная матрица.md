#### обратная матрица - [[матрица]], при умножении на изначальную матрицу получается [[единичная матрица]] 

##### свойства
-  $AA^{-1} = A^{-1}A = E$ 

##### вычислении
- $A^{-1} = \frac{1}{\Delta A}C$, где $\Delta A$ - [[определитель]], $С$ - матрица значений [[дополнительный минор|миноров всех элементов]] 

##### теорема 
- для любой [[вырожденная матрица|вырожденной матрицы]] существует обратная матрица
- **доказательство**
	- $AB = E, AB_j = E_j, j = 1, ..., n$
