#### если непрерывная на отрезке $[-\pi, \pi]$ функция $f(x)$ имеет коэффициенты равные 0, то и сама эта функция тождественно равна 0

##### доказательство
- $$\frac{a_0^2}{2} + \sum(a_m^2 + b_m^2) = \frac{1}{\pi}\int_{-\pi}^{\pi}f^2(x)dx = 0 \Rightarrow f(x) \equiv 0$$

##### замечание 1
- кроме функции тождественно равная 0 не существует непрерывной функции на отрезке $[-\pi, \pi]$, которая была ортогональна ко всем функциям тригонометрической системы 
##### замечание 2
- если две абсолютно интегрируемые функции имеют одни и те же коэффициенты фурье, то их разность равна 0
