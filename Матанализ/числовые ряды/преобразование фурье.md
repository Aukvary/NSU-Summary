#### преобразование фурье - функция $F(z) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} f(u)e^{izu}du$ сопоставляемая функции $f(x)$

##### обозначение
- $f(x) \overset{\mathcal{F}}{\to} F(w)$ $\mathcal{F}$ - преобразование фурье 

##### свойства
- свойство линейности $$\mathcal{F}(\alpha f(t) + \beta g(t)) = \alpha \mathcal{F}(f(t) + \beta \mathcal{F}(g(t)) = \alpha F(w) + \beta G(t)$$$$\mathcal{F}(f(t - t_0)) = e^{iwt_0}F(w)$$
- изменение масштаба $$\mathcal{F} [f(at)] = \frac{1}{|a|}F(\frac{w}{a})$$
- свойство дифференцирования $$\mathcal{F}(f^\prime(t)) = -iw \mathcal{F}(f(t)) = -iw F(w)$$
	- **доказательство:** $$\mathcal{F}(f^\prime(t)) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty f^\prime (u) e^{iwu}du = $$$$= \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty e^{-iwu}df(u) = \underset{= 0}{\underbrace{f(u)e^{iwu}}}|_{-\infty}^\infty - \int_{-\infty}^\infty f(t)(iw)e^{iwt}dt = iw \mathcal{F}[f(t)]$$
- дифференцирование спектра $$\mathcal{F}(tf(t)) = i F^\prime(w)$$
- равенство парсерваля$$\int_{-\infty}^\infty |f(x)|^2 dx = \int_{-\infty}^\infty |F(w)|^2 dw$$$$\int_{-\infty}^\infty f(x)\overline{g(x)}dx = \int_{-\infty}^\infty F(w)\overline{G(w)}dw$$$$<f(x), \overline{g(x)}> = <F(w), \overline{G(w)}>$$
- теорема о свёртке $$(f * g)(t) = \int_{-\infty}^\infty f(\tau) g(t - \tau)d\tau\quad\text{сврёртка f и g}$$$$\mathcal{F}((f * g)(t)) = \sqrt{2\pi} F(w) \cdot G(w)$$
