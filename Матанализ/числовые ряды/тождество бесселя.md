#### вывод
- предположим, что есть какое-то $[a, b]$ на ортогональной плоскости $\{\varphi_m\}, m = 0, 1, 2, ...$, также есть функция $f(x)$ 
- $\sigma_n = c_0\varphi_0 + ... + c_n\varphi_n$, тогда $\Delta_n = \int_a^b [f(x) - \sigma_n(x)]^2dx \to \min$ 
- $$\Delta_n = \int_a^b [f^2(x) - 2f(x)\sigma_n(x) + \sigma^2_n(x)]dx = $$$$\int_a^b f^2(x)dx - 2\sum c_m \int_a^b f(x)\varphi_m(x)dx + \sum c_m^2 \int_a^b \varphi^2_m(x)dx + \underset{\text{= 0 в силу ортогонаольности}}{\underbrace{2\sum_{k < m} c_kc_m \int_a^b\varphi_k\varphi_mdx}}$$
- $$\lambda_m = \int_a^b \varphi_m^2 dx \quad\quad c_m^\prime = \frac{1}{\lambda_m}\int_a^b f(x)\varphi(x)dx$$
- $$\Delta_n = \int_a^bf^2(x)dx - 2\sum \lambda_m c^\prime_m c_m + \sum \lambda_m c^2_m = $$$$\int_a^bf^2(x)dx - \sum\lambda_m c_m^2 + \sum \lambda_m(c_m - c_m^\prime)^2$$
- если $c_m = c_m^\prime$ тогда в пределе можно записать $$\Delta_n = \int_a^b f^2(x)dx - \sum \lambda_m c_m^2\quad\text{ - тождество бесселя}$$
- из него следует неравенство $$\sum \lambda_mc_m^2 \le \int_a^b f^2(x)dx\text{ - неравенство бесселя}$$
- если [[отклонение]] стремится к 0, то $$\sum \lambda_mc_m^2 = \int_a^b f^2(x)dx\text{ - уравнение замкнутости}$$
