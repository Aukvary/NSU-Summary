- рассмотрим непрерывную на $[a;b]$ функцию $f(x)$
- $\{\varphi_m(x)\}$ образуют [[векторное пространство]] $f(x) + g(x)$ 
- можем ввести число $||f(x)|| = \sqrt{\int_a^b f^2(x)dx}$ - `посчетие модуля вектора-норма`
- $f(x) \equiv 0$ 

- $$||f + g|| \le ||f|| + ||g||\quad \text{ - неравенство минковского}$$
- $$||f + g||^2 = \int_a^b (f + g)^2dx = \int_a^b f^2dx + 2\int_a^b fgdx + \int_a^b g^2dx = ||f^2|| + 2\int_a^b fgdx + ||g||^2$$
- $$||f||^2 + 2\int_a^b fgdx + ||g||^2 \le ||f^2|| + 2||f||||g|| + ||g||^2$$
- $$||f \cdot g|| \le ||f|| \cdot ||g|| - \text{невернство коши-буняковского}$$
	- **доказательство** $$\int_a^b (\lambda f + g)^2 \ge 0$$$$\lambda^2\int_a^b f^2dx + 2\lambda\int_a^bfgdx + \int_a^bg^2dx \ge 0$$$$b^2 < ac \Rightarrow (\lambda\int_a^b fgdx)^2 \le \lambda^2\int_a^b f^2 \cdot \int_a^b g^2dx \Rightarrow ||fg|| \le ||f|| \cdot ||g||$$
- $$<f, g> = \int_a^b fgdx$$
- $$\cos \theta = \frac{<f, g>}{||f||\cdot||g||}$$
- $$c_m =\ <f(x), \varphi(x)>\ = \int_a^bf(x)\varphi(x)dx \quad \int_a^b f^2(x)dx = \sum c_m^2 - \text{обобщение теорема пифагора}$$
