#### если члены [[знакопеременный ряд|знакопеременного ряда]] монотонно убывают по абсолютной величине и стремятся к 0 при $n \to \infty$, то ряд сходится 

##### доказательство
- рассмотрим частную сумму конечных слагаемых $$C_{2m} = c_1 - c_2 + ... + c_{2m-1} - c_{2m}$$
- если сгруппируем, они все больше $0$ $C_{2m}$ возрастает $$\underset{> 0}{(c_1 - c_2)} + \underset{> 0}{(c_3 - c_4)} + ...$$
- с другой стороны $$C_{2m} = c_1 - (c_2 - c_3) - (c_4 - c_5) - ...$$
- $C_{2m} < c_1$ - ограничена сверху
- монотонная ограниченная последовательность имеет предел, поэтому сходится 
- теперь рассмотрим нечётную $$C_{2m-1} = \underset{\text{сумма ряда}}{C_{2m}} + \underset{\text{член ряда}}{c_{2m}}$$
- на предел не влияет $\Rightarrow$ сходится
