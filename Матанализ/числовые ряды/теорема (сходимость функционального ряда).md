#### для того, чтобы [[функциональная последовательность]] $\{f_n(x)\}$ имела предел и сходилась к нему равномерное в области $X$, необходимо и достаточно, чтобы для каждого $\varepsilon > 0$ существовало независящие $N$, чтобы $\forall m, n > N$ выполнялось неравенство $|f_n(x) - f_m(x)| < \varepsilon$, $\forall x \in X$ 

##### доказательство
- **необходимость:**
	- предположим, что последовательность сходится равномерно, тогда $$|f_n(x) - f(x)| < \frac{1}{2}\varepsilon \quad |f_m(x) - f(x)| < \frac{1}{2}\varepsilon$$
	- $$|f_n(x) - f_m(x)| = f_n(x) - f(x) + f(x) - f_m(x) \le |f_n(x) - f(x)| + |f(x) - f_m(x)| \le \varepsilon$$
- **достаточность:**
	- зафиксируем $x_0 \ in X$, тк по условию $|f_n(x) - f_m(x)| \le \varepsilon$, то выполняется [[теорема (сходимость ряда)|условие коши]]  
	- значит для каждого $x_0$ существует предел последовательности $\{f_n(x)\}$ $x_0 \in X$ и соответственно существует предел $f(x) = \underset{n, m \to \infty}{\lim} fn(x)\quad |f_n(x) - f_m(x)| \le \varepsilon$ $\forall n  > N, \forall x \in X$, что означает равномерную сходимость
