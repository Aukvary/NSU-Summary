#### [[радиус сходимости]] [[степенной ряд|степенного ряда]] вычисляется по формуле $$\frac{1}{R} = \underset{n\to\infty}{\lim}\sqrt[n]{|a_n|} \quad R = \underset{n\to\infty}{\lim} |\frac{a_n}{a_{n+1}}|, a_n != 0$$

##### доказательство
- следует из признаков сходимости в предельной форме 
- $$J_c = \underset{n\to\infty}{\lim} \sqrt[n]{|a_nx^n|} = |x| \underset{n\to\infty}{\lim} \sqrt[n]{|a_n|} < 1 \Rightarrow |x| < |\frac{1}{\underset{n\to\infty}{\lim} \sqrt[n]{|a_n|}}|$$
- $$J_D = \underset{n\to\infty}{\lim} |\frac{a_{n+1}x^{x^{n+1}}}{x_nx^n}| \le |x| \underset{n\to\infty}{\lim} |\frac{a_{n+1}}{x_n}| < 1 \Rightarrow |x| < \underset{n\to\infty}{\lim} |\frac{a_n}{a_{n+1}}|$$
