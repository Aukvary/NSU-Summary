##### предположение
- график периодикой функции можно представить в виде суммы некоторого числа синусоид 
- $\varphi(t) = A_0 + A_1 \sin(wt + \alpha_1) + A_2\sin(2wt + \alpha_2) + ... = A_0 + \sum  A_n\sin(nwt + a_n)$ 
- $A_n \sin(nwt + \alpha_n) = A_n\sin(nwt)\cos(\alpha_n) + A_n\cos(nwt)\sin(\alpha_n)$ 
- $$f(x) = a_0 + \sum_{n=1}^\infty (a_n \sin(nwt) + b_n \cos(nwt))\text{ - ряд фурье}$$
- $f(x)$ - непрерывная или $k$ непрерывная $x \in [-\pi;\pi]$ 
- $$\int_{-\pi}^\pi f(x)dx = \int a_0dx_{\to 2\pi} + \sum a_n (\int_{-\pi}^\pi \cos(nx)dx + b_n \int_{-\pi}^\pi \sin(nx)dx)_{\to 0} \Rightarrow a_0 = \frac{1}{2\pi} \int_{-\pi}^\pi f(x)dx$$
- $$\int_{-\pi}^\pi f(x)\cos(mx)dx = a_0 \int_{-\pi}^\pi \cos(mx)dx + \sum [a_n \int_{-\pi}^\pi \cos(nx)\cos(mx)dx + b_n \int_{-\pi}^\pi \sin(nx)\cos(mx)dx]$$
- $$\int_{-\pi}^\pi \sin(nx)\cos(mx)dx = \frac{1}{2}\int_{-\pi}^\pi[sin(m+n)x + sin(n+m)x]dx = \pi\text{, при m != n}$$
- $$\int_{-\pi}^\pi \sin(nx)\cos(mx)dx = \frac{1}{2} \int_{-\pi}^\pi [\sin(2nx) + \sin(0)]dx = 0\text{, при m = n}$$
- $$\int_{-\pi}^\pi \cos(nx)\cos(mx)dx = \frac{1}{2} \int_{-\pi}^\pi [\cos(n + m)x + \cos(n - m)x]dx = 0, \text{ при n != m}$$
- $$\frac{1}{2} \int_{-\pi}^\pi [\cos(2nx) + \cos(0)]dx = \pi\text{, при n = m}$$
- $$\begin{cases}a_m = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\cos(mx)dx\\ b_m = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\sin(mx)dx\end{cases}\quad (m = 1, 2, ...)\text{ - коэффициенты фурье}$$
