#### если функция с периодом $2\pi$ кусочно дифференцируема на отрезке $[-\pi, \pi]$, то её [[ряд фурье]] в каждой точке $x$ сходится и имеет сумму $S = \frac{f(x + 0) - f(x - 0)}{2}$, эта сумма равна $f(x_0)$, если в $x_0$ функция непрерывна

##### доказательство
- рассмотрим $f(x) \equiv 1$ 
- $S_n = 1$
- $$1 = \frac{2}{\pi} \int _0^\pi \frac{\sin(n + \frac{1}{2})}{2\sin \frac{t}{2}}dt$$
- $$S_0 = \frac{1}{\pi}\int_0^{\pi} [f(x_0 + 0) + f(x_0 - 0)] \frac{\sin(n + \frac{1}{2})t}{2\sin \frac{t}{2}}dt$$
- $$S_n(x) - S_0 = \frac{1}{\pi}\int_0^\pi [f(x_0 + t) + f(x_0 - t) - f(x_0 + 0) - f(x_0 - 0)]\frac{\sin(n + \frac{1}{2})t}{2\sin \frac{t}{2}}dt + $$$$+[\frac{f(x_0 + t) - f(x_0+ 0)}{t} - \frac{f(x_0 - t) - f(x_0 - 0)}{-t}] \frac{\frac{t}{2}}{\sin(\frac{t}{2})}$$
- $$\underset{t \to \infty}{\lim}\frac{\frac{t}{2}}{\sin(\frac{t}{2})} = 1$$
- $$1.\frac{f(x_0 + t) - f(x_0+ 0)}{t} - \frac{f(x_0 - t) - f(x_0 - 0)}{-t} = 0$$
- $$2. \frac{f(x_0 + t) - f(x_0)}{t} - \frac{f(x_0 - t) - f(x_0)}{-t} = const$$
- $$3. \frac{f(x_0 + t) - f(x_0 - 0)}{t} - \frac{f(x_0 - t) - f(x_0 - 0)}{-t} = const$$
- $f(x)$ нечётная: $$f(x) = -f(-x) \quad \int_{-\pi}^\pi f(x)dx = 0 \quad \int_{-\pi}^\pi f(x)\cos xdx = 0 \Rightarrow f(x) = \sum_{n=1}^\infty b_n \sin(nx)$$
- $f(x)$ чётная: $$f(x) = f(-x) \quad \int_{-\pi}^\pi f(x)dx = 2\int_0^\pi f(x)dx \quad \int_{-\pi}^\pi f(x) \sin(x)dx = 0 \Rightarrow f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty a_n \cos(nx)$$
