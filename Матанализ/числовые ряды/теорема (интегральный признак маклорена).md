#### ряд $\sum f(x)$ сходится, если предел первообразной при $x \to \infty$ конечен, и расходится, если предел бесконечен 

##### доказательство
- предположим, что $F(x) = \int_1^x f(x)dx$ соответственно $a_{n+1}  = f(n + 1) \le f(x) \le f(n) = a_n$ при $n \le x \le n + 1$, ввиду монотонности $a_{n+1} \le \int_n^{n+1} f(x)dx \le a_n$ 
- тогда для ряда $\sum_{n=1}^\infty \int_n^{n+1} f(x)dx$ частичная сумма имеет вид $\int_n^{n+1} f(x)dx = F(n + 1)$
