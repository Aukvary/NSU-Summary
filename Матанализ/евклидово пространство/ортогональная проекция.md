#### ортогональная проекция - [[вектор]] $\lambda \vec e$ для вектора $\vec x$  на прямую, задаваемую вектором $\vec e$ 
![[теорема(представление вектора).excalidraw]]
##### теорема
- [[длинна вектора(евклидово пространство)|длинна]] ортогональной проекции $\vec x$ не превосходит длинны $\lambda \vec e$ 
- **доказательство:**
	- $|\vec x| = (\vec x, \vec x) = (\lambda \vec e, \vec y)$
	- $(\lambda \vec e, \vec y) = (\lambda \vec e, \lambda \vec e) + (\vec e, \vec y) + (\vec e, \vec y) + (\vec y, \vec y)$, $(\vec e, \vec y) = 0 \Rightarrow (\lambda \vec e, \lambda \vec e) + (\vec y, \vec y) = |\lambda \vec e|^2 + |\vec y| \geq |\lambda \vec e|^2 \Rightarrow |\vec x| \geq |\lambda \vec e|$
