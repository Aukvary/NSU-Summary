#### в [[евклидово пространство|евклидовом пространстве]] $\forall \vec x, \vec y: |(\vec x, \vec y)| \leq |\vec x| \cdot |\vec y|$ 

##### доказательство
- пусть $\vec x = \lambda \vec y + \vec z, (\vec z, \vec y) = 0$ 
- $(\vec x, \vec y) = (\lambda \vec y + \vec z, \vec y) = (\lambda \vec y, \vec y) + (\vec z, \vec y) = \lambda(\vec y, \vec y) = \lambda |\vec y|^2$ 
- $|(\vec x, \vec y)| = \lambda |\vec y|^2 = |\lambda \vec y||\vec y| \leq |\vec x||\vec y|$ 

##### следствия 
-  $\cos \varphi = \frac{(\vec x, \vec y)}{|\vec x| \cdot |\vec y|} \lt 1$  
- $(\vec x + \vec y) \leq |\vec x| + |\vec y|$
	- $|\vec x + \vec y|^2 = (\vec x + \vec y)^2 = (\vec x + \vec y, \vec x + \vec y) = (\vec x, \vec x) + 2(\vec x, \vec y) + (\vec y, \vec y) =$ $= |\vec x|^2 + 2(\vec x, \vec y) + |\vec y|^2 \leq |\vec x|^2 + 2|\vec x||\vec y| + |\vec y|^2$  
	- $|\vec x + \vec y|^2 \leq (|\vec x| + |\vec y|)^2 \rightarrow |\vec x + \vec y| \leq |\vec x| + |\vec y|$ 
- $|\vec x + \vec y|^2 = |\vec x|^2 + |\vec y|^2 \Leftrightarrow (\vec x, \vec y) = 0$
