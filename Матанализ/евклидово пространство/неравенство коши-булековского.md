#### неравенство коши-булековского - уравнение вида $|(\vec x, \vec y)| \leq |\vec x| \cdot |\vec y|$, где $\vec x, \vec y$ - [[вектор|векторы]] [[евклидово пространство|евклидова пространства]] 

##### доказательство
- пусть $\vec x = \alpha \vec y + \vec z, (\vec z, \vec y) = 0$ 
- $(\vec x, \vec y) = (\alpha \vec y + \vec z, \vec y) = (\alpha \vec y, \vec y) + (\vec z, \vec y)_{\to 0} = \alpha(\vec y, \vec y) = \alpha|\vec y|^2$ 
- $|(\vec x, \vec y)| = |\alpha\vec y||\vec y| \le |\vec x||\vec y|$, тк $\vec x = \alpha \vec y + \vec z$ 

##### следствие 1
$$\cos\varphi = \frac{(\vec x, \vec y)}{|\vec x||\vec y|} \le 1$$
##### следствие 2
$$|\vec x + \vec y| \le |\vec x| + |\vec y|$$
- **доказательство:**
	-  $$|\vec x + \vec y|^2 = (\vec x +\vec y, \vec x + \vec y) = (\vec x, \vec x) + 2(\vec x, \vec y) + (\vec y, \vec y) = |\vec x|^2 + 2|(\vec x, \vec y)| + |y|^2 \le |\vec x|^2 + 2|\vec x||\vec y| + |\vec y|^2$$
	- $$|\vec x + \vec y|^2 \le (|\vec x| + |\vec y|)^2 \to |\vec x + \vec y| \le |\vec x| + |\vec y|$$
##### следствие 3
$$|\vec x + \vec y|^2 = |\vec x|^2 + |\vec y|^2\text{, если }(\vec x, \vec y)=0\text{ (теорема пифагора)}$$
