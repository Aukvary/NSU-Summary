#### чтобы $y = f(x)$ в точке $x_0$ была [[дифференцируемая функция|дифференцируемой]], необходимо и достаточно, чтобы в этой точке существовала конечная производная $y^\prime = f^\prime(x_0)$ 

##### доказательство
1. необходимость
$$\Delta y = A \Delta x + o(\Delta x)$$
$$\frac{\Delta y}{\Delta x} = A + \frac{o(\Delta x)}{\Delta x}$$
$$\underset{\Delta x \to 0}{\lim}\frac{\Delta y}{\Delta x} = A - \text{предел существует и равен A}$$
2. достаточность
$$\underset{\Delta x \to 0}{\lim}\frac{\Delta y}{\Delta x} = f^\prime(x)$$
$$\frac{\Delta x}{\Delta y} = f^\prime(x_0) + \alpha(\Delta x), \quad \Delta x \to 0 \Rightarrow \alpha(\Delta x) \to 0$$
$$\Delta y = f^\prime(\Delta x) + \alpha(\Delta x) * \Delta x$$
$$o(\Delta x) = \alpha(\Delta x) * \Delta x$$
$$\Delta y = f^\prime(x_0) * \Delta x + o(\Delta x) - \text{определение дифференцируемости}$$

| функция         | производная                                             | дифференциал                                                    |
| --------------- | ------------------------------------------------------- | --------------------------------------------------------------- |
| с               | 0                                                       | $dy = 0 * dx = 0$                                               |
| $x^\mu$         | $\mu x^{\mu -1}$                                        | $dy = \mu x^{\mu - 1}dx$                                        |
| $f(\varphi(t))$ | $y^\prime_t = f^\prime(\varphi(t)) * \varphi^\prime(t)$ | $dy = y^\prime_t dt = y^\prime_x * x^\prime_t dt = y^\prime dx$ |
