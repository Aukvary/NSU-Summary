#### пусть $f(x)$ и $g(x)$ определена на $(a, b]$, существует 2 предела $\underset{x \to a}{\lim} f(x) = 0, \underset{x \to a}{\lim} g(x) = 0$, а также существуют конечные [[производная|производные]] $f^\prime(x)$ и $g^\prime(x) \neq 0$, существует(конечный или нет) предел $\underset{x \to a}{\lim} \frac{f^\prime(x)}{g^\prime(x)} = k$ $\Rightarrow$  $\underset{x \to a}{\lim} \frac{f(x)}{g(x)} = k$ 

##### доказательство
- предложим, что $a$ конечно, тогда $f(a) = g(a) = 0$ => непрерывность $[a, b]$
- по [[Билет 6 (Критерий Коши для последовательностей)|теореме коши]]
	- $$\frac{f(x)}{g(x)} = \frac{f(x) - f(a)\to 0}{g(x) - g(a)\to 0} = \frac{f^\prime(c)}{g^\prime(c)}, \begin{array}{} a < c < x, x \to a\quad c \to a \\ g(c) \neq g(a), \text{ тк } g(c) \neg 0 \end{array}$$
- $$\underset{x \to a}{\lim} \frac{f(x)}{g(x)} = \underset{x \to a}{\lim} \frac{f^\prime(x)}{g^\prime(x)} = k$$
##### замечания
- теорема справедлива при $x \to \pm \infty$  
- правило можно применять повторно
- правило применимо при $\underset{x \to a}{\lim} f(x) = \infty, \underset{x \to a}{\lim} g(x) = \infty$ 
- **доказательство:**
	- допустим $\forall x: f(x) > 0 \land g(x) > 0$ 
	- предположим, что $k$ конечно при $\varepsilon > 0, \delta > 0$ 
	- что при $$\begin{array}{} a < x < a + \delta \\ \delta \leq b - a \end{array}$$ тогда выполняется $$| \frac{f^\prime(x)}{g^\prime(x)} - k|\lt \frac{\varepsilon}{2}$$
	- по [[Билет 6 (Критерий Коши для последовательностей)|формуле коши]] 
		- $$\frac{f(x) - f(a + \delta)}{g(x) - g(a + \delta)} = \frac{f^\prime(c)}{g^\prime(c)}, x < c < a + \delta$$
		- $$\Rightarrow |\frac{f(x) - f(a + \delta)}{g(x) - g(a + \delta)} - k| = \frac{\varepsilon}{2}$$
		- $$\Rightarrow |\frac{f(a + \delta) - g(a + \delta)k}{g(x)}| = \frac{\varepsilon}{2}$$
		- $$\frac{f(x)}{g(x)} < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} < \varepsilon$$
		- $k = +\infty$ 
		- $f^\prime(x) \neq 0 \text{ при } x \to a$
