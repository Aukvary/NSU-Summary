$p(x) = a_0 + a_1x + a_2x^2 + ... + a_n x^n$ 
x = 0
$p^\prime = a_1 + 2a_2x + 3a_3x^2 + ... + na_nx^{n - 1}$
$p^{(2)} = 2a_2 + 6a_3x + ... + n(n - 1) a_n x^{n-2}$ 
$p^{(3)} = 6a_3x + ... + n(n - 1)(n-2)a_n x^{n-3}$ 
$p^{(n)} = 1 * 2 * 3 * ... * na_n$ 
$a_0 = p(0), a_1 = \frac{p^1(0)}{1!}, a_2 = \frac{p^{(2)}(0)}{2!}, a_n \frac{p^{(n)}(0)}{n!}$ 

#### пусть $f(x)$ имеет [[производная|производную]] вплоть до (n - 1) в некоторой окрестности точки $x_0$ и производную [[производные и дифференциалы высших порядков|n-ого порядка]] в самой точке $x_0$, тогда $$p_n(x) = f(x_0) + \frac{f^\prime(x_0)}{1!}(x-x_0) + \frac{f^{\prime\prime}(x_0)}{2!}(x-x_0)^2 + \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n$$
