#### уравнение прямой - $x\cos \alpha + y\sin\alpha - p = 0$ 

##### вывод уравнения
- ![[уравнение прямой(нормальное).excalidraw]]
- $\vec e = \{\cos \alpha, \sin \alpha\}$  
- $\vec{ON} = p$ 
- $x \cos \alpha + y \sin \alpha = p$ 

##### свойства
- связь с [[Матанализ/прямая на плоскости и в пространстве/уравнение прямой(общее)|общим уравнением прямой]] $Ax + By + c = 0$ 
	- $\cos \alpha = \lambda A$
	- $\sin \alpha = \lambda B$ 
	- $-p = \lambda C$ 
	- $\cos^2 \alpha + \sin^2 \alpha = \lambda^2 (A^2 + B^2) = 1$ => $\lambda = \frac{1}{\sqrt{A^2 + B^2}}$ 
	- $\{A, B\}$ [[коллинеарные векторы|коллинеарен]] $\vec e$ 
- если прямая проходит проходит через $O$, то $p$ равен 0 => $x\cos \alpha + y\sin \alpha = 0$ $(\vec{OM}, \vec e) = 0$
