#### пусть [[векторное пространство]] $L$ является суммой некоторых своих подпространств, для того чтобы $L$ было прямой суммой необходимо и достаточно, $\vec x_1 + ... + \vec x_k = 0$ выполнялось только при всех $\vec x_i = 0$ 

##### доказательство
- $\vec w = \vec u_1 + ... + \vec u_k$   -> $\vec 0 = \vec 0_1 + ... +\vec 0_k$ является единственным
- пусть $\vec 0 = \vec 0_1 + ... + \vec 0_k$ единственное представление и пусть для одного вектора есть 2 представления
	- $\vec w = \vec u_1 + ... + \vec u_k$
	- $\vec{w^\prime} = \vec v_1 + ... + \vec v_k$ 
- $\vec w - \vec{w^\prime} = (\vec u_1 - \vec v_1) + ... + (\vec u_k - \vec v_k)$
