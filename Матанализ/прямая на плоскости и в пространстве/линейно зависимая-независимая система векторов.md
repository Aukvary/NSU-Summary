#### линейно зависимая система векторов - система [[вектор|векторов]] $\vec a_1, ..., \vec a_n$, где существует хотя бы одна нетривиальная комбинация этих векторов равна $\vec 0$ в противном случае система **линейно независима** 

##### свойства
- если $\vec 0 \in \vec a_1, ..., \vec a_n$ => система линейно зависима 
- если среди $\vec a_1, ..., \vec a_n$ некоторые векторы образуют линейно зависимую систему, то вся система линейно зависима 
- если система векторов линейно зависима, то по крайней мере один из векторов является линейной комбинацией остальных
- система состоящая из двух векторов, тогда и только тогда линейно зависима, где векторы [[коллинеарные векторы|коллинеарны]] 
- система состоящая из трёх векторов векторов тогда и тогда тогда линейно зависима, когда эти 3 вектора [[компланарные векторы|компланарны]]
