#### если все коэффициенты из [[уравнение прямой(общее)|общего уравнения прямой]] одной прямой равны коэффициентам другой прямой, то они совпадают 

##### доказательство 
- пусть $d: Ax + By + C = 0$ $l: A^\prime x + B^\prime y + C^\prime + 0$ 
- если эти условия выполняются, то уравнения подобны
- 1 случай $B = B^\prime = 0$:
$$
	\begin{cases}
	Ax + C = 0 \\
	A^\prime x + C = 0
	 \end{cases} \Rightarrow 
	 \begin{cases}
	x = -\frac{C}{A^\prime} \\
	x = -\frac{C}{A^\prime}
	 \end{cases} \Rightarrow \frac{A}{A^\prime} = \frac{C}{C^\prime}$$
- 2 случай, будем смотреть относительно $x = 0$
$$
\begin{cases}
A0 + By + C = 0 \\
A^\prime0 + B^\prime y + C^\prime = 0 
\end{cases} 

\Rightarrow 

y = -\frac{C}{B} = -\frac{C^\prime}{B^\prime}
\Rightarrow 
\frac{C}{C^\prime} = \frac{B}{B^\prime}
\Rightarrow 
\frac{C}{C^\prime} = \frac{B}{B^\prime} \frac{A}{A^\prime}
$$
