#### уравнение прямой - $A(x-x_0) + B(y - y_0) = 0$, где $\vec n = \{A, B\}$ - [[нормаль]] и $\vec l = \{x - x_0, y - y_0\}$ - [[направляющий вектор прямой]] $l$ 
![[уравнение прямой(нормаль).excalidraw]]
##### вывод уравнения
- пусть $A(x_0, y_0)$ - произвольная точка на прямой $l$, вектор $\vec n$ - [[нормаль]] 
- зададим вторую произвольную точку $M(x, y)$ на прямой $l$ 
- => $\vec {AM} = \vec r - \vec r_0$ 
- по [[скалярное произведение#свойства|свойству скалярного произведения]] $(\vec{AM}, \vec n) = 0$, тк $\vec n$ - перпендикулярен
- => $(\vec r - \vec r_0, \vec n) = A(x - x_0) + B(y - y_0) = 0$
