#### угол между векторами - наименьший из двух смежных углов, образованный этим прямыми 

##### способ нахождения
- пусть прямые заданы [[уравнение прямой(общее)|общим уравнением прямой]] 
$$
\begin{cases}
	l_1: A_1x + B_1y + C_1 = 0 \\
	l_2: A_2x + B_2y + C_2 = 0
\end{cases}
$$
- тогда [[направляющие вектора|направляющие]] будут иметь вид $\vec l_1 = \{B_1, A_1\}$ и $\vec l_2 \{B_2, A_2\}$, пусть угол между прямыми равен $\varphi$ => $$\cos \varphi = \frac{(\vec l_1; \vec l_2)}{|\vec l_1||\vec l_2|} = \frac{|A_1A_2 + B_1B_2|}{\sqrt{A_1^2 + B_1^2}\sqrt{A_2^2 + B_2^2}}$$
- пусть прямые заданы [[уравнение прямой(угловой коэффициент)|уравнением с угловым коэффициентом]] 
$$
\begin{cases}
	l_1: y = k_1 x + b_1 \\
	l_2: y = k_2 x + b_2
\end{cases}
$$
- если переписать его в общем виде
$$
\begin{cases}
	l_1: -k_1 x + y - b_1 \\
	l_2: -k_2 x + y + b_2
\end{cases}
$$
- мы получаем направляющие буду иметь вид $\vec l_1 = \{1, k_1\}$ и $\vec l_2 = \{1, k_2\}$, пусть угол между прямыми равен $\varphi$ => $$\cos \varphi = \frac{|k_1k_2 + 1|}{\sqrt{k_1^2 + 1}\sqrt{k_2^2 + 1}}$$
- тк $k_1 = \tan\varphi_1$, $k_2 = \tan \varphi_2$, пусть угол между прямыми равен $\varphi$, тогда $\varphi = |\varphi_2 - \varphi_1|$ =>$$|\tan(\varphi_2 - \varphi_1)| = |\frac{\tan\varphi_2 - \tan\varphi_1}{1 + \tan\varphi_1\varphi_2}| \Rightarrow \tan \varphi = |\frac{k_2 - k_1}{1 + k_1k_2}|$$
