#### кватернионы - числа вида $q = a + bi + cj + dk$, где $a, b, c, d \in \mathbb{R}$, а $i, j,k$ - мнимые единицы

##### свойства
- $i^2 = j^2 = k^2 = -1$ 
- $ij = k, jk = i, ki = j$
- $ji = -k, kj = -i, ik = -j$ 

##### операции
- сложение
- умножение
- сопряжение $\overline{a + bi+cj+dk} = a - bi - cj - dk$ 
- модуль $|a + bi + cj + dk| = \sqrt{a^2 + b^2 + c^2 + d^2}$, $|q| = q \bar q = \bar q q = \sqrt{a^2 + b^2 + c^2 + d^2} = \sqrt{|q|^2}$  
- деление $q \neq 0 \quad q^{-1} = \frac{\bar q}{|q|^2} \quad qq^{-1} = 1$ 
- $\cases{\vec u = u_1i + u_2j + u_3k \\ \vec v_1i + v_2j + v_3k}, \vec u \vec v = \overset{\text{скаляр}}{\overline{-(u_1v_1 + u_2v_2 + u_3v_3)}} + \overset{\text{вектор}}{\overline{(u_2v_3 - u_3v_2)i + (u_3v_1 - u_1v_3)j + (u_1v_2 - u_2v_1)k}}$ 
- поворот
	- $q = \cos(\frac{\theta}{2}) + \sin(\frac{\theta}{2})(n_xi + n_yj + n_zk)$, $\{n_x, n_y, n_z\}$ - единицы в оси поворота $\theta$ - угол
	- $v = xi + yj + zk \quad v^\prime = qv\bar q$
