##### замечания
- сумма вектором не зависит от выбора точки $O$
- $\vec 0 + \vec a = \vec a + \vec 0 = \vec a$, $\vec a + (-\vec a) = (-\vec a) + \vec a = \vec 0)$ 
- $\vec a + \vec b = \vec b + \vec a$ - коммутативность
- $(\vec a + \vec b) + \vec c = \vec a + (\vec b + \vec c)$ - ассоциативность
