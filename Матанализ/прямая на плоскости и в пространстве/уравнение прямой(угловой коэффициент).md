#### уравнение прямой - $y = kx + b$ 

##### вывод уравнения
- из [[уравнение прямой(каноническое)|канонического уравнения прямой имеем]] $\frac{x - x_0}{m} = \frac{y - y_0}{n}$ => $y - y_0 = \frac{n}{m}(x - x_0)$, где $\frac{n}{m} = k$ - [[условный коэффициент направляющих]] 
- отсюда можно вывести прямой, проходящую через точку $A(x_0, y_0)$ уравнение вида $y = y(x_0) + k(x - x_0)$ => $y = y_0 + kx + kx_0$, обозначим $y_0 - kx_0$ за $b$ 
- => получим уравнение $y = kx + b$
