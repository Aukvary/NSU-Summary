#### скалярное произведение - $|\vec u_1| * |\vec u_2| = \cos(\varphi)$, где $\vec u_1, \vec u_2$ векторы, а $\varphi$ угол между векторами 

##### обозначения
- $(\vec u_1, \vec u_2)$

##### свойства
- $(\vec u_1, \vec u_2) = (\vec u_2, \vec u_1)$ 
- $(\vec u_1, \vec u_2) = 0 \Leftrightarrow \vec u_1 \bot \vec u_2$  
- $(\vec u, \vec u) = |(\vec u_1, \vec u_2)|^2$ 
- пусть $\vec u_1 = \{x_1, y_1\}, \vec u_2 = \{x_2, y_2\}$ => $\vec u_1 = x_1 * \vec e_1 + y_1 * \vec e_2, \vec u_2 = x_2 * \vec e_1 + y_1 * e_2$, тогда 
	- $(\vec u_1, \vec u_2) = (x_1 * \vec e_1 + y_1 * \vec e_2, x_2 * \vec e_1 + y_1 * e_2) =$ 
	- $= x_1x_2(\vec e_1, \vec e_1) + y_1x_2(\vec e_2, \vec e_1) + x_1y_2(\vec e_1, \vec e_1) + y_1y_2(\vec e_1, \vec e_1)$, тк $(\vec e_1, \vec e_2) = 0, (\vec e_1, \vec e_2) = 1$ =>
	- $(\vec u_1, \vec u_2) = x_1x_y + y_1y_2$ 
- $\cos (\varphi) = \frac{(\vec u_1, \vec u_2)}{|\vec u_1| |\vec u_2|} = \frac{x_1x_2 + y_1y_2}{\sqrt{x_1^2 + y_1^2} * \sqrt{x_1^2 + y_1^2}}$
