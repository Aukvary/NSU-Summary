#### задача - деление отрезков в заданной пропорции 
$$\frac{\vec{AM}}{\vec{MB}} = \frac{\alpha}{\beta} \to \frac{\vec {A_xM_x}}{\vec {M_xBx}} = \frac{\alpha}{\beta} = \frac{x - x_1}{x_2 - x}$$
$$\alpha(x_2 - x) = \beta(x - x_1)$$
$$x = \frac{\alpha x_2 + \beta x_1}{\alpha + \beta}$$
- аналогично для $y$ $$y = \frac{\alpha y_2 + \beta y_1}{\alpha + \beta}$$
