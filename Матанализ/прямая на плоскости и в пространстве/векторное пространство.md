#### векторное пространство - непустое множество [[вектор|векторов]] принадлежащих этому множеству, если выполняется несколько условий

##### условия 
- существует правило сопоставляющее двум любым двум векторам $\vec x$ и $\vec y$ третий вектор называемый суммой и обозначаемый $\vec x + \vec y$ 
- определено правило сопоставляющее каждому вектору $\vec x$ и каждому числу $\lambda$ новый вектор называемый произведением $\lambda$ и $\vec x$ и обозначается $\lambda \vec x$ 
- эти операции должны удовлетворять следующим условиям 
	- $\vec x + \vec y = \vec y + \vec x$
	- $(\vec x + \vec y) + \vec y = \vec x + (\vec y + \vec y)$ 
	- $\exists \vec 0 \in L: \forall \vec x\in l: x + \vec x + \vec 0 = \vec x$ 
	- $\forall \vec x \in L: \exists \vec{-x} \in L: \vec x + \vec{-x} = \vec 0$ 
	- $\forall \lambda, \vec x, \vec y: \lambda(\vec x + \vec y) = \lambda\vec x + \lambda\vec y$ 
	- $\forall \alpha, \beta, \vec x: (\alpha + \beta)\vec x = \alpha\vec x + \beta\vec x$ 
	- $\alpha(\beta \vec x) = (\alpha\beta)\vec x$
	- $\forall \vec x: 1\vec x = \vec x \land 0\vec x = \vec 0$
