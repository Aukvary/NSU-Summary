####  рациональная функция $\frac{P(x)}{Q(x)}$, где $P(x)$ и $Q(x)$ - многочлены 

##### 1
$$\int \frac{A}{x-a}dx = A\ln|x-a| + c$$

##### 2
$$\int \frac{A}{(x-a)^k}dx = -\frac{A}{k-1}\cdot\frac{1}{(x-a)^{k-1}}+c$$

##### 3
$$\int \frac{Mx + N}{x^2 + px +q} = \int \frac{Mx + N}{(x + \frac{p}{2})^2 + (q - \frac{p^2}{4})}$$
$$\cases{x + \frac{p}{2} = t, dx = dt\\ x^2 + px + q = t^2 + a^2 \\ a^2 = q - \frac{p^2}{4}}\quad Mx + N = Mt + (N - \frac{Mp}{2})$$
$$\int \frac{Mx + N}{x^2 + px + q} = \int \frac{Mt + (N -\frac{Mp}{2})}{t^2 + a^2}dt = \frac{M}{2} \int \frac{2tdt}{t^2 + a^2} + (N - \frac{Mp}{2})\int \frac{dt}{t^2 + a^2} =$$
$$= \frac{M}{2}\ln(t^2+a^2) + \frac{1}{a}(N-\frac{Mp}{2})\arctan(\frac{x+\frac{p}{2}}{a}) + c $$
##### 4
$$\int \frac{Mx + N}{(x^2 + px + q)^m} = \int \frac{Mt + (N - \frac{Mp}{2})}{(t^2 + a^2)^m}dt = \frac{M}{2}\int \frac{2tdt}{(t^2+a^2)^m} + (N - \frac{Mp}{2})\int \frac{dt}{(t^2+a^2)^m}$$
$$\int \frac{2tdt}{(t^2+a^2)^m} = -\frac{1}{m-1}\cdot\frac{1}{(t^2 + a^2)^{m-1}} + c$$
$$I_n = \int \frac{dx}{(x^2+a^2)^n} = \frac{x}{(x^2 + a^2)^n} + 2n\int \frac{x^2dx}{(x^2+a^2)^{n+1}}$$
$$\int \frac{x^2}{(x^2 + a^2)^{n+1}}dx = \int \frac{x^2 + a^2 - a^2}{(x^2 + a^2)^{n+1}}dx = \int \frac{dx}{(x^2 + a^2)^n} - a^2 \int \frac{dx}{(x^2+a^2)^{n+1}} = I_n - a^2 I{n+1}$$
$$I_n = \frac{x}{(x^2+a^2)^n} + 2nI_n - 2na^2I_{n+1}$$
$$I_{n+1} = \frac{1}{2na^2} \cdot \frac{x}{(x^2+a^2)^n} + \frac{2n-1}{2na^2}I_n$$
$$I_1 = \frac{1}{a}\arctan(\frac{x}{a}) + c$$
