#### пусть $f(x)$ непрерывна на $[a, b]$ и $F(x)$ любая её первообразная, тогда $\int_a^b f(x)dx = F(b) - F(a)$ 

##### доказательство
- пусть есть $\Phi(x) = \int_a^bf(x)dx \to \Phi^\prime(x) = f(x)$ 
- $F(x) = \Phi(x) + c$ - любая другая [[первообразная]]
- $\Phi(a) = \int_a^a f(x)dx = 0$ 
- $F(a) = c$ 
- $\Phi(x) = F(x) - F(a)$
- $\int_a^b f(x)dx = \Phi(b) = F(b) - F(a)$ 

##### следствие 1
- пусть $\int_a^b f(x)dx$, $f(x)$ непрерывна на $[a, b]$, $x  = \varphi(t)$ 
- при условии
	- $\varphi(t)$ определена и непрерывна на $[\alpha, \beta]$ и её значения не выходят за $[a, b]$, когда $t \in [\alpha, \beta]$ 
	- $\varphi(\alpha) = a, \varphi(\beta) = b$ 
	- на отрезке $[\alpha, \beta]$ существует непрерывная производная $\varphi^\prime(t)$ 
- тогда $\int_a^bf(x)dx = \int_\alpha^\beta f(\varphi(t))\varphi^\prime(t)dt$
