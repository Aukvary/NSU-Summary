#### суммы дорбу - суммы $s = \sum m_i\Delta x_i\quad S = \sum M_i \Delta x_i$, где $m_i, M_i$ - точные границы $f(x)$ на $[x_i, x_{i+1}]$ 

##### свойства
- $s \le \sigma \le S$
- если к имеющимся точкам деления добавить новые точки, то имеющаяся сумма может только возрасти, а верхняя границы уменьшиться $$x^\prime, x_k < x^\prime < x_{k+1} \quad \overline M \le M_k, \overline{\overline M} < M_k$$ $$\overline M_k(x^\prime - x_k) \le M_k \quad \overline{\overline M}_k(x_{k+1} \le M_k(x_{k+1}-x^\prime)$$$$\overline M(x^\prime - x_k) + \overline{\overline M}(x_{k+1}- x^\prime) \le M_k(x_{k+1} - x_k)$$$$S^\prime \le S$$
- каждая нижняя сумма дорбу не превосходит каждой верхней суммы, хотя бы отвечающие другому разбиению промежутков 
	- **доказательство:**
	- 1. $s_1, S_1$, 2. $s_2, S_2$   $s_3, S_3$
		- $s_1 \le S_2 ?$ 
		- $s_1 \le s_3$
		- $S_3 \le S_2$ 
	- $s_1 \le s_3 \le S_3 \le S_2 \to s_1 \le S_2$ 

##### замечание
- если $J_* = \sup{s} \quad J^* = \inf{S}$, то $s \le J_* \le J^* \le S$
