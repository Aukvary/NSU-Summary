#### если в некотором конечном или бесконечном замкнутом линией промежутке $X$ $F(x)$  - [[первообразная]] для $f(x)$, то для $F(x) + c$, где $c$ - какая-то константа, тоже будет являться первообразной, обратно каждая функция первообразная для $F(x)$ может быть ...

##### доказательство
- $[F(x) + c]^\prime = F^\prime(x) + c^\prime_{\to 0} = F^\prime(x) = f(x)$ 
- $[\Phi(x) - F(x)]^\prime = \Phi^\prime(x) - F^\prime(x) = f(x) - f(x) = 0$ 
	- $x_1 < c < x_2$ 
	- $\psi(x_2) - \psi(x_1) = \psi^\prime(c)(x_2 - x_1)$ 
	- $\psi^\prime(c) = 0 \quad \psi(x_2) = \psi(x_1)$
	- $\psi(x) = const$ 

##### следствие
- $\int f(x) \, dx = F(x) \to d\int f(x) \, dx = dF(x) = f(x)dx$ 
- $\int F^\prime(x) \, dx = F(x) + c \to \int F(x) \, dx = F(x) + c$
