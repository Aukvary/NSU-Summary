##### пусть $\forall n \in N \exists [a_n, b_n] a_n \leq b_n$ => задана последовательность отрезков $\{[a_n, b_n]_{n=1}^{\infty}\}$ 
##### пусть $\{[a_n, b_n]_{n=1}^{\infty}\}$ последовательность вложенных отрезков, если $\forall n \in N: a_n <= a_{n+1} <= b_{n+1} <= _n$ 
#### теорема кантора - $\forall$ последовательность вложенных отрезков имеют хотя бы одну общую точку. $\exists c \in R c \in [a_n, b_n] \forall n \in N$ <=> $\underset{n=1}{\overset{\infty}{\cap}} [a_n, b_n] \neq \varnothing$ 

#### доказательство:
- $A = [a_1, a_2, ..., a_n, ...]$ - множество левых концов 
- $B = [b_1, b_2, ..., b_n, ...]$ - множество правых концов 
- покажем, что A расположен левее множества B 
	- $\forall n \in N$ и $\forall m \in N$ $a_n \leq a_m$ 
	- пусть $n \leq m$ 
	- тк отрезки вложены => $a_n \leq a_{n + 1}$ => $a_n \leq ... \leq a_m$ 
	- => $[a_m, b_m]$ - отрезок => $a_n ... \leq a_m \leq b_m$ 
- $\exists c \in R:$ $c$ разделяет A и B => $\forall m \in N c \leq b_m$ и $\forall n \in N a_n \leq c$ 
- => $c \in [a_n, b_n] \forall n \in N$ => $c \in \underset{n=1}{\overset{\infty}{\cap}} [a_n, b_n]$
