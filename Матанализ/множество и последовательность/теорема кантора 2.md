#### теорема кантора - стягивающаяся последовательность имеет одну общую точку $\underset{n=1}{\overset{\infty}{\cap}} [a_n, b_n]$ 
#### доказательство:
- тк стягивающаяся последовательность - это последовательность вложенных отрезков, то $\underset{n=1}{\overset{\infty}{\cap}} [a_n, b_n] \neq \varnothing$ 
- предположим, что пересечение состоит более чем из одной точки $c$ и $c'$ => $c \in \underset{n=1}{\overset{\infty}{\cap}} [a_n, b_n]$ и $c' \in \underset{n=1}{\overset{\infty}{\cap}} [a_n, b_n]$ 
- пусть $c \leq c'$ => $[c, c'] \subset \underset{n=1}{\overset{\infty}{\cap}} [a_n, b_n]$ по лемме архимеда $\mathbb{N}$ не ограниченно сверху => $\exists l > \frac{1}{|c - c'|}$ => $\exists n(l): |a_{n(l)} - b_{n(l)}| < \frac{1}{l} < |c-c'|$  
- но $[c, c'] \subset [a_{n(l)}, b_{n(l)}]$ => $|c - c'| \leq a_{n(l)}, b_{n(l)}|$ - противоречие => $c = c'$ 

- $\forall n\in\mathbb{N}\ [c, c'] \subset [a_n, b_n]$ => $0 < |c' - c| \leq|b_n - a_n|$ 
- тк $\forall ε > 0 \in \mathbb{N} \ \exists N \in \mathbb{N}: \forall n > N |a_n - b_n| < ε$ из определения [[последовательность стягивающихся отрезков|стягивающихся отрезков]] пусть $ε = \frac{|c' - c|}{2}$ 
- => $|a_n - b_n| < \frac{|c - c'|}{2}$
