```C
void insert_sort(int* nums, int len) {
	for (int i = 1; i < len; i++) { //проходимся по всем элементам начиная с 1-ого
		for (int j = i; j > 0 && nums[j] < nums[j - 1]; j--) { //проверяем, что текущий элмент не меньше предыдущего
			swap(&nums[j], &nums[j - 1]); //если текущий меньше , то меняем их местами
		}
	}
}
```
##### асимптотика
- скорость - O($N^2$ )
- память - O(1)
