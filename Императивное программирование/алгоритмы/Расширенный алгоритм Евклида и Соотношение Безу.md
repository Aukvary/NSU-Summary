##### теорема
- $\forall a, b: \exists x, y = ax + by = \text{НОД}(a, b)$, коэффициенты $x, y$ называются коэффициентами Безу 
- **применение:**
	- если $\text{НОД}(a, m) = 1$, то $ax = 1(\text{mod } m)$ имеет решение 

```C
int extended_gcd(int a, int b, int* x, int* y) {
	if (a == 0) {
		*x = 0;
		*y = 1;
		return b;
	}
	
	int x1;
	int y1;
	
	int gcd = extended_gcd(b % a, a, &x1, &x2);
	
	*x = y1 - (b / a) * x1;
	*y = x1;
	
	return gcd;
}
```
