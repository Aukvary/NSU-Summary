- позволяет быстро находить все простые числа в диапазоне `[2; n]` 
- как работает:
	- начинаем с 2, вычёркиваем все числа, которые кратные ей
	- берём следующее невычеркнутое число и вычёркиваем все кратные ему и тд 
```C
int N;
bool not_prime[N] = {0};

void init() {
	for (int p = 2; p*p <= N; p++) {
		if (not_prime[p]) continue;
		
		for (int i = p*p; i <= N; i += p) {
			not_prime[i] = true;
		}
	}
}
```
