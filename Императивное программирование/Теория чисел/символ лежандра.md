#### символ лежандра($\frac{a}{p}$) - функция, которая показывает, является ли число $a$ [[квадратичный вычет по модулю|квадратичным вычетом]] по простому $p$ $$(\frac{a}{p}) = \begin{cases}1, \text{если ф - квадратичный вычет и a}\neq 0 (\text{mod } p)\\-1, \text{если a - квадратичный невычет}\\ 0, \text{если} a \equiv 0(\text{mod } p)\end{cases}$$
